1二维随机变量

1.1 联合分布函数

$$F(x,y) = P(X \leq x, \; Y \leq y)$$ $$\text{性质：} \;\; F(-\infty, y) = F(x, -\infty) = 0, \;\; F(+\infty, +\infty) = 1$$ $$F \text{ 关于 } x, y \text{ 分别单调不减、右连续}$$

1.2 离散型联合分布律

$$p_{ij} = P(X = x_i, \; Y = y_j) \geq 0, \quad \sum_i \sum_j p_{ij} = 1$$ $$F(x,y) = \sum_{x_i \leq x} \sum_{y_j \leq y} p_{ij}$$

1.3 连续型联合密度

$$f(x,y) \geq 0, \;\; \iint f(x,y)\,dxdy = 1$$ $$F(x,y) = \int_{-\infty}^y \int_{-\infty}^x f(u,v)\,dudv$$ $$f(x,y) = \frac{\partial^2 F}{\partial x \partial y} \qquad P((X,Y) \in D) = \iint_D f(x,y)\,dxdy$$

2边际分布与条件分布

2.1 边际分布

$$\text{离散：} \;\; p_{i\cdot} = \sum_j p_{ij}, \quad p_{\cdot j} = \sum_i p_{ij}$$ $$\text{连续：} \;\; f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,dy, \quad f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,dx$$
注意：边际分布由联合分布唯一确定，但反过来不成立——不同的联合分布可以有相同的边际分布。

2.2 条件分布

$$\text{离散：} \;\; P(X = x_i \;|\; Y = y_j) = \frac{p_{ij}}{p_{\cdot j}} \quad (p_{\cdot j} > 0)$$ $$\text{连续：} \;\; f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)} \quad (f_Y(y) > 0)$$

2.3 二维正态分布

$$(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$$ $$X \sim N(\mu_1, \sigma_1^2), \;\; Y \sim N(\mu_2, \sigma_2^2)$$ $$\text{Cov}(X,Y) = \rho \sigma_1 \sigma_2 \qquad \rho_{XY} = \rho$$ $$X, Y \text{ 独立} \;\Leftrightarrow\; \rho = 0$$
特别注意：对于二维正态分布，不相关 $\Leftrightarrow$ 独立。这仅对正态分布成立，一般分布不成立。

3随机变量的独立性

3.1 定义

$$X, Y \text{ 独立} \;\Leftrightarrow\; F(x,y) = F_X(x) \cdot F_Y(y)$$ $$\Leftrightarrow\; p_{ij} = p_{i\cdot} \cdot p_{\cdot j} \;\text{（离散）}$$ $$\Leftrightarrow\; f(x,y) = f_X(x) \cdot f_Y(y) \;\text{（连续，几乎处处）}$$ $$\Leftrightarrow\; f_{X|Y}(x|y) = f_X(x) \;\text{（条件 = 无条件）}$$

3.2 独立的推论

  • $X, Y$ 独立 $\Rightarrow$ $g(X), h(Y)$ 独立（任意可测函数）
  • $X, Y$ 独立 $\Rightarrow$ $E(XY) = E(X)E(Y)$
  • $X, Y$ 独立 $\Rightarrow$ $D(X \pm Y) = D(X) + D(Y)$
  • 二维正态：独立 $\Leftrightarrow$ $\rho = 0$

4二维随机变量函数的分布

4.1 $Z = X + Y$ 的分布（卷积公式）

$$f_Z(z) = \int_{-\infty}^{+\infty} f(x, z - x)\,dx$$ $$\text{若独立：} \;\; f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x)\,dx$$
积分限：由 $f_X(x)$ 和 $f_Y(z-x)$ 的非零区域确定。例如均匀分布 $[0,1]$：需 $0 \leq x \leq 1$ 且 $0 \leq z-x \leq 1$。

4.2 $M = \max(X,Y)$ 和 $N = \min(X,Y)$

设 $X, Y$ 独立：

$$F_M(z) = F_X(z) \cdot F_Y(z) \quad \text{（都}\leq z\text{）}$$ $$F_N(z) = 1 - [1 - F_X(z)][1 - F_Y(z)] \quad \text{（至少一个}\leq z\text{）}$$

推广到 $n$ 个独立同分布变量：

$$F_{\max}(z) = [F(z)]^n \qquad F_{\min}(z) = 1 - [1 - F(z)]^n$$

4.3 正态分布的可加性

$$X \sim N(\mu_1, \sigma_1^2), \; Y \sim N(\mu_2, \sigma_2^2), \; \text{独立} \;\Rightarrow\; X + Y \sim N(\mu_1 + \mu_2, \; \sigma_1^2 + \sigma_2^2)$$ $$aX + bY \sim N(a\mu_1 + b\mu_2, \; a^2\sigma_1^2 + b^2\sigma_2^2)$$

5数学期望

5.1 定义

$$\text{离散：} \;\; E(X) = \sum_i x_i p_i \qquad \text{连续：} \;\; E(X) = \int_{-\infty}^{+\infty} x\,f(x)\,dx$$

5.2 随机变量函数的期望

$$E(g(X)) = \sum_i g(x_i) p_i \quad\text{或}\quad E(g(X)) = \int g(x) f(x)\,dx$$ $$E(g(X,Y)) = \iint g(x,y) f(x,y)\,dxdy$$
关键：求 $E(g(X))$ 不需要先求 $g(X)$ 的分布，直接用 $X$ 的分布对 $g(x)$ 积分即可。

5.3 期望的性质

$$E(C) = C \qquad E(CX) = CE(X) \qquad E(X + C) = E(X) + C$$ $$E(X \pm Y) = E(X) \pm E(Y) \qquad E\!\left(\sum a_i X_i\right) = \sum a_i E(X_i)$$ $$X, Y \text{ 独立} \;\Rightarrow\; E(XY) = E(X)E(Y)$$

6方差

6.1 定义与计算公式

$$D(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$$ $$\sigma(X) = \sqrt{D(X)} \;\text{（标准差）}$$

6.2 性质

$$D(C) = 0 \qquad D(CX) = C^2 D(X) \qquad D(X + C) = D(X)$$ $$D(X \pm Y) = D(X) + D(Y) \pm 2\,\text{Cov}(X,Y)$$ $$X, Y \text{ 独立} \;\Rightarrow\; D(X \pm Y) = D(X) + D(Y)$$

7协方差与相关系数

7.1 协方差

$$\text{Cov}(X,Y) = E[(X - EX)(Y - EY)] = E(XY) - E(X)E(Y)$$ $$\text{Cov}(X,X) = D(X) \qquad \text{Cov}(aX+b, cY+d) = ac\,\text{Cov}(X,Y)$$ $$\text{Cov}(X+Y, Z) = \text{Cov}(X,Z) + \text{Cov}(Y,Z)$$ $$D(X \pm Y) = D(X) + D(Y) \pm 2\,\text{Cov}(X,Y)$$

7.2 相关系数

$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)} \sqrt{D(Y)}} \in [-1, 1]$$ $$|\rho| = 1 \;\Leftrightarrow\; P(Y = aX + b) = 1 \;\text{（线性关系）}$$ $$\rho = 0 \;\Leftrightarrow\; \text{不相关} \;\Leftrightarrow\; \text{Cov}(X,Y) = 0 \;\Leftrightarrow\; E(XY) = E(X)E(Y)$$

7.3 独立与不相关的关系

$$\text{独立} \;\Rightarrow\; \text{不相关} \quad (\text{但反之一般不成立})$$ $$\text{二维正态：独立} \;\Leftrightarrow\; \text{不相关} \;\Leftrightarrow\; \rho = 0$$
关键区分：独立比不相关更强。独立要求所有关系都不存在，不相关只要求线性关系不存在。仅对正态分布，两者等价。
