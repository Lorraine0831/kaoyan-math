1偏导数

1.1 定义

设 $z = f(x,y)$，固定 $y$ 对 $x$ 求导：

$$f_x(x_0,y_0) = \lim_{\Delta x \to 0}\frac{f(x_0+\Delta x, y_0)-f(x_0,y_0)}{\Delta x}$$ $$f_y(x_0,y_0) = \lim_{\Delta y \to 0}\frac{f(x_0, y_0+\Delta y)-f(x_0,y_0)}{\Delta y}$$
理解：偏导数就是"其他变量视为常数"时对某一变量的导数，反映函数沿坐标轴方向的变化率。

1.2 二阶偏导数与混合偏导

$$f_{xx} = \frac{\partial^2 f}{\partial x^2}, \quad f_{yy} = \frac{\partial^2 f}{\partial y^2}, \quad f_{xy} = \frac{\partial^2 f}{\partial x\,\partial y}, \quad f_{yx} = \frac{\partial^2 f}{\partial y\,\partial x}$$

1.3 混合偏导相等的条件

$$\text{若 } f_{xy} \text{ 和 } f_{yx} \text{ 在某区域内连续} \;\Rightarrow\; f_{xy} = f_{yx}$$
实操：考试中遇到的初等函数一般都满足此条件，混合偏导可交换求导顺序。

2全微分

2.1 定义

若 $\Delta z = f(x+\Delta x, y+\Delta y) - f(x,y)$ 可表示为

$$\Delta z = A\,\Delta x + B\,\Delta y + o(\rho), \quad \rho = \sqrt{\Delta x^2+\Delta y^2}$$

则称 $f$ 在 $(x,y)$ 可微，记 $dz = A\,dx + B\,dy$。若可微，则 $A = f_x$，$B = f_y$。

2.2 可微的必要条件

$$f \text{ 可微} \;\Rightarrow\; f_x \text{ 和 } f_y \text{ 均存在}$$
注意：偏导数存在不一定可微！需验证 $\lim_{\rho\to 0}\frac{\Delta z - f_x\Delta x - f_y\Delta y}{\rho} = 0$。

2.3 可微的充分条件

$$f_x \text{ 和 } f_y \text{ 连续} \;\Rightarrow\; f \text{ 可微}$$

2.4 全微分公式

$$dz = \frac{\partial f}{\partial x}dx + \frac{\partial f}{\partial y}dy$$ $$\text{三元：} \quad dz = f_x\,dx + f_y\,dy + f_z\,dz$$

2.5 全微分形式不变性

无论 $u, v$ 是自变量还是中间变量，$z = f(u,v)$ 的全微分形式不变：

$$dz = f_u\,du + f_v\,dv$$

2.6 近似计算

$$\Delta z \approx dz = f_x\,\Delta x + f_y\,\Delta y$$ $$f(x_0+\Delta x, y_0+\Delta y) \approx f(x_0,y_0) + f_x(x_0,y_0)\Delta x + f_y(x_0,y_0)\Delta y$$

3多元复合函数求导

3.1 链式法则（一般情形）

$$z = f(u,v), \quad u = \varphi(x,y), \quad v = \psi(x,y)$$ $$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial f}{\partial v}\frac{\partial v}{\partial x}$$ $$\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u}\frac{\partial u}{\partial y} + \frac{\partial f}{\partial v}\frac{\partial v}{\partial y}$$

3.2 全导数（中间变量为一元函数）

$$z = f(u(t), v(t)) \;\Rightarrow\; \frac{dz}{dt} = \frac{\partial f}{\partial u}\frac{du}{dt} + \frac{\partial f}{\partial v}\frac{dv}{dt}$$

3.3 复合关系中既有 $x$ 又有中间变量含 $x$

$$z = f(x, u(x,y)) \;\Rightarrow\; \frac{\partial z}{\partial x} = \frac{\partial f}{\partial x} + \frac{\partial f}{\partial u}\frac{\partial u}{\partial x}$$
符号注意：这里 $\frac{\partial f}{\partial x}$ 是把 $u$ 视为常数时对 $x$ 的偏导，与 $\frac{\partial z}{\partial x}$ 含义不同！

3.4 画树状图辅助记忆

技巧：画出函数依赖关系树：$z \to u \to x$ 和 $z \to v \to x$，从 $z$ 到 $x$ 的每条路径对应一项乘积，所有路径之和即为 $\frac{\partial z}{\partial x}$。

4隐函数求导

4.1 一个方程的情形

$$F(x,y) = 0 \;\Rightarrow\; \frac{dy}{dx} = -\frac{F_x}{F_y} \quad (F_y \neq 0)$$ $$F(x,y,z) = 0 \;\Rightarrow\; \frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z} \quad (F_z \neq 0)$$

4.2 方程组的情形

设 $\begin{cases} F(x,y,z) = 0 \\ G(x,y,z) = 0 \end{cases}$ 确定 $y=y(x), z=z(x)$，则

$$\frac{dy}{dx} = -\frac{\begin{vmatrix} F_x & F_z \\ G_x & G_z \end{vmatrix}}{\begin{vmatrix} F_y & F_z \\ G_y & G_z \end{vmatrix}} = -\frac{F_xG_z - F_zG_x}{F_yG_z - F_zG_y}, \quad \frac{dz}{dx} = -\frac{\begin{vmatrix} F_y & F_x \\ G_y & G_x \end{vmatrix}}{\begin{vmatrix} F_y & F_z \\ G_y & G_z \end{vmatrix}}$$
实操技巧：对方程组两边直接对 $x$ 求导（视 $y,z$ 为 $x$ 的函数），解线性方程组，比记行列式更可靠。

4.3 隐函数存在定理

若 $F(x_0,y_0)=0$，$F_y$ 在某邻域连续且 $F_y(x_0,y_0) \neq 0$，则在 $(x_0,y_0)$ 附近唯一确定连续可微的隐函数 $y=y(x)$。

5方向导数与梯度

5.1 方向导数

函数 $f(x,y)$ 沿方向 $\vec{l} = (\cos\alpha, \cos\beta)$ 的方向导数：

$$\frac{\partial f}{\partial \vec{l}} = f_x\cos\alpha + f_y\cos\beta$$ $$\text{三元：} \quad \frac{\partial f}{\partial \vec{l}} = f_x\cos\alpha + f_y\cos\beta + f_z\cos\gamma$$

5.2 梯度

$$\text{梯度：} \quad \vec{\nabla f} = (f_x, \; f_y) \quad \text{（三元：}\;(f_x, f_y, f_z)\text{）}$$ $$\text{方向导数} = \vec{\nabla f} \cdot \vec{l^0}$$

5.3 梯度的意义

  • 方向导数的最大值 $= |\nabla f| = \sqrt{f_x^2 + f_y^2}$，方向沿梯度方向
  • 方向导数最小值 $= -|\nabla f|$，方向沿负梯度方向
  • 梯度方向是函数增长最快的方向
  • 梯度垂直于等值线（面）

6多元函数的极值

6.1 极值的必要条件

若 $f$ 在 $(x_0,y_0)$ 有偏导且取极值，则

$$f_x(x_0,y_0) = 0, \quad f_y(x_0,y_0) = 0$$

满足此条件的点称为驻点。极值点一定是驻点或偏导不存在的点。

6.2 极值的充分条件（判别法）

设 $f_x(x_0,y_0)=0$，$f_y(x_0,y_0)=0$，令 $A = f_{xx}$，$B = f_{xy}$，$C = f_{yy}$，$\Delta = AC - B^2$：

条件结论
$\Delta > 0$，$A > 0$极小值
$\Delta > 0$，$A < 0$极大值
$\Delta < 0$不是极值（鞍点）
$\Delta = 0$不能确定，需用其他方法
记忆：$\Delta = AC-B^2$ 就像一个"判别式"，$\Delta>0$ 才有极值，再看 $A$ 的正负决定极大极小。

7条件极值与拉格朗日乘数法

7.1 拉格朗日乘数法

目标：在约束 $\varphi(x,y) = 0$ 下求 $f(x,y)$ 的极值。构造拉格朗日函数：

$$\mathcal{L}(x,y,\lambda) = f(x,y) + \lambda\,\varphi(x,y)$$ $$\text{解方程组：} \begin{cases} \mathcal{L}_x = f_x + \lambda\varphi_x = 0 \\ \mathcal{L}_y = f_y + \lambda\varphi_y = 0 \\ \mathcal{L}_\lambda = \varphi(x,y) = 0 \end{cases}$$

7.2 多约束条件

若有两个约束 $\varphi_1=0, \varphi_2=0$，则引入两个乘子：

$$\mathcal{L} = f + \lambda_1\varphi_1 + \lambda_2\varphi_2$$ $$\text{对 } x,y,z,\lambda_1,\lambda_2 \text{ 求偏导令其为零，解方程组}$$

7.3 条件极值的几何理解

直觉：在约束曲面上移动，当目标函数的梯度与约束曲面的法向量平行时（即 $\nabla f = -\lambda\nabla\varphi$），达到极值。

8多元微分学的几何应用

8.1 空间曲线的切线与法平面

参数方程 $\vec{r}(t) = (x(t), y(t), z(t))$，在 $t=t_0$ 处：

$$\text{切向量：} \;\vec{T} = (x'(t_0), y'(t_0), z'(t_0))$$ $$\text{切线方程：} \;\frac{x-x_0}{x'(t_0)} = \frac{y-y_0}{y'(t_0)} = \frac{z-z_0}{z'(t_0)}$$ $$\text{法平面方程：} \;x'(t_0)(x-x_0) + y'(t_0)(y-y_0) + z'(t_0)(z-z_0) = 0$$

8.2 曲面的切平面与法线

隐式方程 $F(x,y,z) = 0$，在 $(x_0,y_0,z_0)$ 处：

$$\text{法向量：} \;\vec{n} = (F_x, F_y, F_z)\Big|_{(x_0,y_0,z_0)}$$ $$\text{切平面方程：} \;F_x(x-x_0) + F_y(y-y_0) + F_z(z-z_0) = 0$$ $$\text{法线方程：} \;\frac{x-x_0}{F_x} = \frac{y-y_0}{F_y} = \frac{z-z_0}{F_z}$$

显式方程 $z = f(x,y)$：

$$\text{法向量：} \;\vec{n} = (f_x, f_y, -1) \quad \text{或} \quad (-f_x, -f_y, 1)$$ $$\text{切平面：} \;z - z_0 = f_x(x-x_0) + f_y(y-y_0)$$

8.3 方向导数与切平面的关系

联系：曲面 $z=f(x,y)$ 在 $(x_0,y_0)$ 处的切平面方程 $z = z_0 + f_x\Delta x + f_y\Delta y$ 本质上就是全微分 $dz = f_x\,dx + f_y\,dy$ 的几何表现。方向导数就是切平面上沿给定方向的斜率。
