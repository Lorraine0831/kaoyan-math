1特征值与特征向量

1.1 定义

设 $A$ 为 $n$ 阶方阵，若存在数 $\lambda$ 和非零向量 $\vec{\alpha}$ 使

$$A\vec{\alpha} = \lambda\vec{\alpha}$$

则称 $\lambda$ 为 $A$ 的特征值，$\vec{\alpha}$ 为对应于 $\lambda$ 的特征向量。

1.2 求法

  1. 求特征方程 $|\lambda I - A| = 0$ 的根，得特征值 $\lambda_i$
  2. 对每个 $\lambda_i$，解齐次方程组 $(\lambda_i I - A)\vec{x} = \vec{0}$，得特征向量

1.3 特征值的性质

$$\sum_{i=1}^n \lambda_i = \text{tr}(A) = \sum a_{ii} \qquad \prod_{i=1}^n \lambda_i = |A|$$ $$\lambda_{A^T} = \lambda_A \qquad \lambda_{A^k} = \lambda^k \qquad \lambda_{A^{-1}} = \frac{1}{\lambda} \qquad \lambda_{A^*} = \frac{|A|}{\lambda}$$ $$\lambda_{f(A)} = f(\lambda) \qquad \lambda_{A+kI} = \lambda + k$$
关键结论：
  • $\lambda = 0 \Leftrightarrow |A| = 0$（$A$ 不可逆）
  • 不同特征值对应的特征向量线性无关
  • $\lambda$ 是 $k$ 重根时，对应特征向量个数（几何重数）$\leq k$（代数重数）

2相似矩阵

2.1 定义

$$A \sim B \;\Leftrightarrow\; \exists\, P \text{ 可逆}, \;\; P^{-1}AP = B$$

2.2 相似的必要条件

$$A \sim B \;\Rightarrow\; \begin{cases} |A| = |B| \\ \text{tr}(A) = \text{tr}(B) \\ r(A) = r(B) \\ \lambda_A = \lambda_B \\ |\lambda I - A| = |\lambda I - B| \end{cases}$$
注意：以上是必要条件，不充分。两个矩阵满足以上所有条件也不一定相似。

3矩阵的对角化

3.1 可对角化的条件

$$A \text{ 可对角化} \;\Leftrightarrow\; \exists\, P \text{ 可逆}, \;\; P^{-1}AP = \Lambda$$ $$\Leftrightarrow\; A \text{ 有 } n \text{ 个线性无关的特征向量}$$ $$\Leftrightarrow\; \text{每个 } k_i \text{ 重特征值有 } k_i \text{ 个线性无关的特征向量}$$ $$\Leftrightarrow\; \text{每个特征值的几何重数} = \text{代数重数}$$

3.2 对角化步骤

  1. 求所有特征值 $\lambda_1, \ldots, \lambda_n$（含重根）
  2. 对每个 $\lambda_i$ 求特征向量
  3. 若有 $n$ 个线性无关的特征向量 $\vec{p}_1, \ldots, \vec{p}_n$，令 $P = (\vec{p}_1, \ldots, \vec{p}_n)$
  4. 则 $P^{-1}AP = \text{diag}(\lambda_1, \ldots, \lambda_n)$

3.3 判断能否对角化

核心：对每个 $k$ 重特征值 $\lambda$，检查 $r(\lambda I - A)$。若 $r(\lambda I - A) = n - k$（即有 $k$ 个线性无关特征向量），则该特征值不阻碍对角化。所有特征值都满足此条件 $\Rightarrow$ 可对角化。

4实对称矩阵

4.1 核心性质

$$\text{实对称矩阵的特征值全为实数}$$ $$\text{不同特征值对应的特征向量正交}$$ $$\text{实对称矩阵一定可正交对角化}$$ $$\exists\, Q \text{ 正交矩阵}, \;\; Q^{-1}AQ = Q^TAQ = \Lambda$$

4.2 正交对角化步骤

  1. 求特征值
  2. 求特征向量
  3. 对重根对应的多个特征向量做施密特正交化
  4. 所有特征向量单位化
  5. 以正交单位特征向量为列构造 $Q$，则 $Q^TAQ = \Lambda$
注意：不同特征值对应的特征向量天然正交，不需要正交化，只需单位化。只有同一特征值对应的多个特征向量之间才需要正交化。

5二次型

5.1 定义与矩阵表示

含 $n$ 个变量的二次齐次多项式：

$$f = \sum_{i,j} a_{ij}x_i x_j = \vec{x}^T A \vec{x}$$

其中 $A$ 为对称矩阵（$a_{ij} = a_{ji}$），称为二次型的矩阵。$r(A) = r(f)$ 称为二次型的秩。

5.2 矩阵表示要点

  • 交叉项 $x_i x_j$（$i \neq j$）的系数 $a_{ij}$ 拆成 $a_{ij} = a_{ji} = \frac{\text{系数}}{2}$
  • 平方项 $x_i^2$ 的系数直接作为 $a_{ii}$

6二次型的标准形与规范形

6.1 标准形

$$f = d_1 y_1^2 + d_2 y_2^2 + \cdots + d_n y_n^2$$

6.2 化标准形的方法

方法一：正交变换法

$$\text{求 } A \text{ 的特征值 } \lambda_i \;\to\; \text{正交矩阵 } Q \;\to\; \vec{x} = Q\vec{y}$$ $$f = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2$$

特点：标准形系数是特征值，结果唯一。

方法二：配方法

步骤：逐步将含某变量的所有项配成完全平方，每次消去一个变量。若无平方项，先做变换 $x_i = y_i + y_j, x_j = y_i - y_j$ 制造平方项。

方法三：合同变换法

$$\begin{pmatrix} A \\ I \end{pmatrix} \xrightarrow{\text{同型行列变换}} \begin{pmatrix} \Lambda \\ C \end{pmatrix}$$ $$\vec{x} = C\vec{y}, \quad f = \vec{y}^T \Lambda \vec{y}$$

6.3 规范形与惯性定理

$$f = y_1^2 + \cdots + y_p^2 - y_{p+1}^2 - \cdots - y_{p+q}^2$$ $$p + q = r(f), \quad p - q = \text{符号差}$$ $$\textbf{惯性定理：} \;\; p, q \text{ 由 } f \text{ 唯一确定}$$
关键：不同的化简方法得到的标准形系数可能不同，但正负惯性指数 $p, q$ 唯一确定。

7正定二次型

7.1 定义

$$f = \vec{x}^T A \vec{x} \text{ 正定} \;\Leftrightarrow\; \forall\, \vec{x} \neq \vec{0}, \;\; f > 0$$

7.2 判定方法

$$\text{正定} \;\Leftrightarrow\; A \text{ 的特征值全为正}$$ $$\Leftrightarrow\; A \text{ 的各阶顺序主子式全为正}$$ $$\Leftrightarrow\; p = r = n \;\text{（正惯性指数 = 秩 = } n\text{）}$$ $$\Leftrightarrow\; A \text{ 合同于 } I_n$$

7.3 顺序主子式判据（最常用）

$$\Delta_1 = a_{11} > 0$$ $$\Delta_2 = \begin{vmatrix} a_{11} & a_{12} \\ a_{21} & a_{22} \end{vmatrix} > 0$$ $$\vdots$$ $$\Delta_n = |A| > 0$$
注意：是顺序主子式（左上角的 $k\times k$ 子式），不是任意主子式。各阶顺序主子式同时为正才正定。

7.4 合同关系

$$A \simeq B \;\Leftrightarrow\; \exists\, C \text{ 可逆}, \;\; C^TAC = B$$ $$\text{实对称矩阵 } A \simeq B \;\Leftrightarrow\; p_A = p_B \text{ 且 } q_A = q_B \;\Leftrightarrow\; \text{正负惯性指数分别相等}$$

等价、相似、合同的关系：

关系定义不变量
等价$PAQ = B$秩
相似$P^{-1}AP = B$特征值、迹、行列式、秩
合同$C^TAC = B$秩、正负惯性指数
联系：正交相似 $\Rightarrow$ 相似且合同。实对称矩阵的相似 $\Leftrightarrow$ 合同。
