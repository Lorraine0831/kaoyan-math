在线使用： 本站已部署至 GitHub Pages，所有页面支持 MathJax 数学公式渲染（需联网加载），并自动适配暗色模式。

复习建议

  1. 通读：按章节顺序阅读知识点页面，理解每个概念和定理
  2. 背诵：重点背诵各科「必背公式」速查表
  3. 提速：精读「做题提速常用等式/技巧」，特别是等价无穷小和泰勒展开
  4. 刷题：做真题时遇到不熟的知识点回到对应章节查阅
