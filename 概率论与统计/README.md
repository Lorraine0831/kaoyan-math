# 概率论与数理统计复习资料

考前系统复习专用，涵盖知识点、必背公式、做题提速技巧。

## 目录结构

```
概率论与统计/
├── index.html                  ← 导航首页
├── README.md
├── assets/
│   └── style.css
├── 知识点/
│   ├── 01_概率与一维随机变量.html
│   ├── 02_二维随机变量与数字特征.html
│   └── 03_大数定律与数理统计.html
└── 公式速查/
    ├── 必背公式.html
    └── 做题提速常用技巧.html
```

## 使用方式

1. 浏览器打开 `index.html` 进入导航首页
2. 所有页面支持 MathJax 公式渲染（需联网加载）
3. 支持暗色模式（跟随系统）
