# 高等数学复习资料

考前系统复习专用，涵盖知识点、必背公式、做题提速常用等式。

## 目录结构

```
高数/
├── index.html                  ← 导航首页（推荐从这里开始）
├── README.md                   ← 本文件
├── assets/
│   └── style.css               ← 通用样式（含暗色模式）
├── 知识点/                      ← 8 章知识点 HTML 页面
│   ├── 01_极限与连续.html
│   ├── 02_一元微分学.html
│   ├── 03_一元积分学.html
│   ├── 04_多元微分学.html
│   ├── 05_多元积分学.html
│   ├── 06_常微分方程.html
│   ├── 07_无穷级数.html
│   └── 08_向量代数与空间解析几何.html
└── 公式速查/
    ├── 必背公式.html            ← 全部核心公式速查表
    └── 做题提速常用等式.html     ← 等价无穷小、泰勒展开、凑微分等
```

## 使用方式

1. 在浏览器中打开 `index.html` 进入导航首页
2. 点击卡片进入对应章节或公式速查
3. 所有页面支持 MathJax 数学公式渲染，需联网加载
4. 支持暗色模式（跟随系统设置自动切换）

## 复习建议

- **通读**：按章节顺序阅读知识点页面
- **背诵**：重点看「必背公式」速查表
- **提速**：精读「做题提速常用等式」，特别是等价无穷小差的形式
- **刷题**：做真题时遇到不熟的知识点回到对应章节查阅
