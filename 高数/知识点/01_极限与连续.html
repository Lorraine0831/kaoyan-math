1函数基础

1.1 函数定义

设 $D$ 为非空数集，若存在确定的法则 $f$，使得对每个 $x \in D$，有唯一的 $y$ 与之对应，则称 $f$ 为定义在 $D$ 上的函数，记作

$$y = f(x), \quad x \in D$$

$D$ 称为定义域，$R_f = \{f(x) \mid x \in D\}$ 称为值域。

1.2 函数的特性

特性定义几何特征
有界性$\exists M>0$，$\forall x\in D$，$|f(x)|\leq M$图像夹在 $y=\pm M$ 之间
单调性$x_1$x_1从左到右上升/下降
奇偶性奇：$f(-x)=-f(x)$

1.3 反函数

若 $y=f(x)$ 是 $D$ 到 $R_f$ 的一一映射，则存在反函数 $x = f^{-1}(y)$，满足

$$f^{-1}(f(x)) = x \quad (x\in D), \qquad f(f^{-1}(y)) = y \quad (y\in R_f)$$ $$y = f(x) \text{ 与 } y = f^{-1}(x) \text{ 的图像关于 } y=x \text{ 对称}$$ $$\text{严格单调函数的反函数存在，且保持同方向单调性}$$

1.4 复合函数

设 $y = f(u)$ 定义域为 $D_f$，$u = g(x)$ 值域为 $R_g$，若 $R_g \subseteq D_f$，则

$$y = f(g(x)), \quad x \in D_g$$

称为由 $f$ 和 $g$ 复合而成的复合函数，记作 $f \circ g$。

由基本初等函数经有限次四则运算和有限次复合构成的、能用一个解析式表示的函数称为初等函数。初等函数在其定义域内是连续的。

考试要点：判断一个函数是否为初等函数，看能否用基本初等函数通过四则运算和复合得到。绝对值函数 $|x| = \sqrt{x^2}$ 是初等函数；分段函数（如 $f(x)=\max(x,x^2)$）一般不是初等函数。

2数列极限

定义（$\varepsilon$-$N$ 语言）

设 $\{a_n\}$ 为数列，若存在常数 $A$，使得对任意 $\varepsilon > 0$，存在 $N \in \mathbb{N}^+$，当 $n > N$ 时 $|a_n - A| < \varepsilon$，则称数列收敛于 $A$，记作

$$\lim_{n \to \infty} a_n = A \quad \text{或} \quad a_n \to A \;(n\to\infty)$$
直觉理解：当 $n$ 足够大时，$a_n$ 可以任意接近 $A$。$N$ 的大小取决于 $\varepsilon$：$\varepsilon$ 越小，需要的 $N$ 越大。

收敛数列的性质

  • 唯一性：收敛数列的极限唯一
  • 有界性：收敛数列必有界（有界是收敛的必要条件，非充分）
  • 保号性：若 $\lim a_n = A > 0$，则存在 $N$，当 $n > N$ 时 $a_n > 0$
  • 保不等式：若 $a_n \leq b_n$ 且 $\lim a_n, \lim b_n$ 均存在，则 $\lim a_n \leq \lim b_n$
  • 子列收敛：若 $\{a_n\}$ 收敛于 $A$，则其任一子列也收敛于 $A$

3函数极限

3.1 $\varepsilon$-$\delta$ 定义

设 $f(x)$ 在 $x_0$ 的某去心邻域有定义。若存在 $A$，使得对任意 $\varepsilon > 0$，存在 $\delta > 0$，当 $0 < |x - x_0| < \delta$ 时 $|f(x) - A| < \varepsilon$，则

$$\lim_{x \to x_0} f(x) = A$$

3.2 单侧极限

$$\text{左极限：} \lim_{x \to x_0^-} f(x) = f(x_0^-) \qquad \text{右极限：} \lim_{x \to x_0^+} f(x) = f(x_0^+)$$ $$\lim_{x \to x_0} f(x) = A \;\Leftrightarrow\; f(x_0^-) = f(x_0^+) = A$$
关键：极限存在的充要条件是左右极限都存在且相等。分段函数在分界点处必须用单侧极限判断。

3.3 自变量趋于无穷

$$\lim_{x \to \infty} f(x) = A \;\Leftrightarrow\; \lim_{x \to +\infty} f(x) = \lim_{x \to -\infty} f(x) = A$$

3.4 水平渐近线

若 $\lim_{x \to +\infty} f(x) = A$ 或 $\lim_{x \to -\infty} f(x) = A$，则 $y = A$ 是 $f(x)$ 的水平渐近线。

3.5 铅直渐近线

若 $\lim_{x \to x_0} f(x) = \infty$（含单侧），则 $x = x_0$ 是 $f(x)$ 的铅直渐近线。

4极限的性质与运算

4.1 极限的四则运算

设 $\lim f(x) = A$，$\lim g(x) = B$，则：

$$\lim[f \pm g] = A \pm B \qquad \lim[f \cdot g] = AB \qquad \lim\frac{f}{g} = \frac{A}{B}\;(B \neq 0)$$ $$\lim[cf(x)] = cA \qquad \lim[f(x)]^n = A^n$$

4.2 复合函数极限

若 $\lim_{x \to x_0} g(x) = u_0$（且 $g(x) \neq u_0$），$\lim_{u \to u_0} f(u) = A$，则

$$\lim_{x \to x_0} f(g(x)) = A$$

4.3 两个重要极限

$$\lim_{x \to 0} \frac{\sin x}{x} = 1 \qquad\qquad \lim_{x \to 0}(1+x)^{1/x} = e \quad\left(\text{或 } \lim_{n\to\infty}\!\left(1+\frac{1}{n}\right)^n = e\right)$$
推广形式：只要 $\square \to 0$，就有 $\frac{\sin\square}{\square} \to 1$，$(1+\square)^{1/\square} \to e$。例如 $\lim_{x\to 0}\frac{\sin 3x}{3x}=1$。

4.4 极限存在准则

  • 夹逼准则：若 $g(x) \leq f(x) \leq h(x)$，且 $\lim g = \lim h = A$，则 $\lim f = A$
  • 单调有界准则：单调递增有上界的数列（或递减有下界）必有极限
  • 柯西收敛准则：$\{a_n\}$ 收敛 $\Leftrightarrow$ 对任意 $\varepsilon > 0$，存在 $N$，当 $m,n > N$ 时 $|a_m - a_n| < \varepsilon$

5无穷小与无穷大

5.1 定义

  • 无穷小：$\lim f(x) = 0$，称 $f(x)$ 为无穷小量
  • 无穷大：$\lim f(x) = \infty$，称 $f(x)$ 为无穷大量
  • 关系：在 $f(x) \neq 0$ 的条件下，$f$ 为无穷大 $\Leftrightarrow$ $\frac{1}{f}$ 为无穷小

5.2 无穷小的比较

设 $\alpha \to 0$，$\beta \to 0$：

关系定义记法
同阶$\lim \frac{\beta}{\alpha} = c \neq 0$$\beta = O(\alpha)$
等价$\lim \frac{\beta}{\alpha} = 1$$\beta \sim \alpha$
高阶$\lim \frac{\beta}{\alpha} = 0$$\beta = o(\alpha)$
低阶$\lim \frac{\beta}{\alpha} = \infty$—
$k$ 阶$\lim \frac{\beta}{\alpha^k} = c \neq 0$$\beta$ 是 $\alpha$ 的 $k$ 阶无穷小

5.3 等价无穷小替换定理

设 $\alpha \sim \alpha'$，$\beta \sim \beta'$，且 $\lim \frac{\beta'}{\alpha'}$ 存在，则

$$\lim \frac{\beta}{\alpha} = \lim \frac{\beta'}{\alpha'}$$
使用规则：等价无穷小只能对乘除因子替换。加减中的无穷小一般不能直接替换——因为可能因低阶相消而露出更高阶项，此时需用泰勒展开。

5.4 常用等价无穷小（$x \to 0$）

$$\sin x \sim \tan x \sim \arcsin x \sim \arctan x \sim \ln(1+x) \sim e^x-1 \sim x$$ $$1-\cos x \sim \tfrac{1}{2}x^2 \qquad (1+x)^a - 1 \sim ax \qquad a^x-1 \sim x\ln a$$ $$x-\sin x \sim \tfrac{1}{6}x^3 \qquad \tan x - x \sim \tfrac{1}{3}x^3 \qquad x-\ln(1+x) \sim \tfrac{1}{2}x^2$$

6连续性

6.1 连续的定义

$f(x)$ 在 $x_0$ 处连续，当且仅当满足以下三个条件：

  1. $f(x_0)$ 有定义
  2. $\lim_{x \to x_0} f(x)$ 存在
  3. $\lim_{x \to x_0} f(x) = f(x_0)$
$$\lim_{x \to x_0} f(x) = f(x_0) \;\Leftrightarrow\; \lim_{\Delta x \to 0}[f(x_0+\Delta x) - f(x_0)] = 0$$

6.2 左连续与右连续

$$\text{左连续：} \lim_{x \to x_0^-} f(x) = f(x_0) \qquad \text{右连续：} \lim_{x \to x_0^+} f(x) = f(x_0)$$ $$f \text{ 在 } x_0 \text{ 连续} \;\Leftrightarrow\; f \text{ 在 } x_0 \text{ 左连续且右连续}$$

6.3 间断点分类

类型条件典型例子
第一类·可去左右极限存在且相等，但 $\neq f(x_0)$ 或 $f(x_0)$ 无定义$f(x)=\frac{\sin x}{x}$，$x=0$
第一类·跳跃左右极限存在但不相等$f(x)=\text{sgn}(x)$，$x=0$
第二类·无穷至少一侧极限为 $\infty$$f(x)=\frac{1}{x}$，$x=0$
第二类·振荡至少一侧极限不存在且非 $\infty$$f(x)=\sin\frac{1}{x}$，$x=0$

6.4 连续函数的运算

  • 连续函数的和、差、积、商（分母非零）仍连续
  • 连续函数的复合函数仍连续
  • 初等函数在其定义域内连续

7闭区间上连续函数的性质

7.1 最大值最小值定理

$$f \in C[a,b] \;\Rightarrow\; f \text{ 在 } [a,b] \text{ 上有最大值和最小值}$$

7.2 有界性定理

$$f \in C[a,b] \;\Rightarrow\; f \text{ 在 } [a,b] \text{ 上有界}$$

7.3 零点定理（重要！）

$$f \in C[a,b], \quad f(a) \cdot f(b) < 0 \;\Rightarrow\; \exists\, c \in (a,b), \;\; f(c) = 0$$
应用：证明方程 $f(x)=0$ 在某区间内有根。只需找两个点使函数值异号即可。

7.4 介值定理

$$f \in C[a,b], \quad \mu \text{ 介于 } f(a) \text{ 和 } f(b) \text{ 之间} \;\Rightarrow\; \exists\, c \in [a,b], \;\; f(c) = \mu$$

7.5 一致连续性

若 $f \in C[a,b]$，则 $f$ 在 $[a,b]$ 上一致连续（康托尔定理）。

区别：连续是逐点的概念，一致连续是整体的概念。开区间 $(a,b)$ 上的连续函数不一定一致连续。
