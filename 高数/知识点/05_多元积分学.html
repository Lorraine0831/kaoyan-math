1二重积分

1.1 定义

$$\iint_D f(x,y)\,d\sigma = \lim_{\lambda\to 0}\sum_{i=1}^n f(\xi_i,\eta_i)\,\Delta\sigma_i$$

当 $f$ 在有界闭区域 $D$ 上连续时，二重积分存在。

1.2 直角坐标系下的计算

X 型区域（先 $y$ 后 $x$）：$D = \{(x,y) \mid a \leq x \leq b,\; y_1(x) \leq y \leq y_2(x)\}$

$$\iint_D f\,dxdy = \int_a^b\left[\int_{y_1(x)}^{y_2(x)} f(x,y)\,dy\right]dx$$

Y 型区域（先 $x$ 后 $y$）：$D = \{(x,y) \mid c \leq y \leq d,\; x_1(y) \leq x \leq x_2(y)\}$

$$\iint_D f\,dxdy = \int_c^d\left[\int_{x_1(y)}^{x_2(y)} f(x,y)\,dx\right]dy$$

1.3 极坐标系下的计算

令 $x = r\cos\theta$，$y = r\sin\theta$：

$$\iint_D f(x,y)\,dxdy = \iint_D f(r\cos\theta, r\sin\theta)\,r\,dr\,d\theta$$
注意：面积元素 $d\sigma = r\,dr\,d\theta$，不要漏 $r$！

极坐标适用场景：

  • 积分域为圆、圆环、扇形或其一部分
  • 被积函数含 $x^2+y^2$
  • 被积函数含 $\arctan(y/x)$（即 $\theta$）

1.4 二重积分的对称性

$$D \text{ 关于 } y \text{ 轴对称：} \iint_D f\,d\sigma = \begin{cases} 0 & f \text{ 关于 } x \text{ 为奇} \\ 2\iint_{D_1} f\,d\sigma & f \text{ 关于 } x \text{ 为偶} \end{cases}$$

同理，关于 $x$ 轴对称、关于原点对称也有类似的对称性简化。

轮换对称性：若 $D$ 关于 $y=x$ 对称（即 $x,y$ 可交换），则 $\iint_D f(x,y)\,d\sigma = \iint_D f(y,x)\,d\sigma$，特别地 $\iint_D x\,d\sigma = \iint_D y\,d\sigma$。

1.5 二重积分的几何意义

  • $f \geq 0$ 时，$\iint_D f\,d\sigma$ 表示以 $D$ 为底、$z=f(x,y)$ 为顶的曲顶柱体体积
  • $f \equiv 1$ 时，$\iint_D d\sigma = D$ 的面积

2三重积分

2.1 直角坐标系

先一后二（投影法）：$\Omega$ 投影到 $xy$ 面得 $D_{xy}$，$z$ 从 $z_1(x,y)$ 到 $z_2(x,y)$：

$$\iiint_\Omega f\,dV = \iint_{D_{xy}}\left[\int_{z_1(x,y)}^{z_2(x,y)} f\,dz\right]dxdy$$

先二后一（截面法）：用 $z=z$ 截 $\Omega$ 得截面 $D_z$：

$$\iiint_\Omega f\,dV = \int_c^d\left[\iint_{D_z} f(x,y,z)\,dxdy\right]dz$$
选法：若被积函数只含 $z$（或 $z$ 的函数），用先二后一更方便。

2.2 柱坐标

令 $x = r\cos\theta$，$y = r\sin\theta$，$z = z$：

$$\iiint_\Omega f\,dV = \iiint_\Omega f(r\cos\theta, r\sin\theta, z)\,r\,dz\,dr\,d\theta$$
注意：体积元素 $r\,dz\,dr\,d\theta$，不要漏 $r$。

适用场景：积分域含旋转对称轴（圆柱、圆锥、旋转抛物面），被积函数含 $x^2+y^2$。

2.3 球坐标

令 $x = \rho\sin\varphi\cos\theta$，$y = \rho\sin\varphi\sin\theta$，$z = \rho\cos\varphi$：

$$\iiint_\Omega f\,dV = \iiint_\Omega f(\rho,\varphi,\theta)\,\rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$$
注意：体积元素 $\rho^2\sin\varphi\,d\rho\,d\varphi\,d\theta$，三个因子都不能漏。其中 $\varphi$ 是与 $z$ 轴正方向的夹角（$0\leq\varphi\leq\pi$），$\theta$ 是 $xy$ 面上的方位角。

适用场景：积分域为球、球壳或锥面与球面围成，被积函数含 $x^2+y^2+z^2$。

2.4 三重积分的对称性

与二重积分类似：

  • 奇偶对称性：$\Omega$ 关于某坐标面对称，$f$ 关于该变量为奇 $\Rightarrow$ 积分为 $0$
  • 轮换对称性：$\Omega$ 关于 $x,y,z$ 的轮换对称 $\Rightarrow \iiint x\,dV = \iiint y\,dV = \iiint z\,dV$
举例：若 $\Omega$ 是球体 $x^2+y^2+z^2\leq R^2$，则 $\iiint_\Omega x\,dV = \iiint_\Omega y\,dV = \iiint_\Omega z\,dV = \iiint_\Omega (x+y+z)\,dV / 3$。

3曲线积分

3.1 第一类曲线积分（对弧长）

$$\int_L f(x,y)\,ds$$

参数方程 $x=x(t), y=y(t)$，$\alpha \leq t \leq \beta$：

$$\int_L f\,ds = \int_\alpha^\beta f(x(t),y(t))\sqrt{x'^2+y'^2}\,dt$$

注意：积分限必须 $\alpha < \beta$（从下限到上限，与方向无关），因为弧长元素 $ds \geq 0$。

3.2 第二类曲线积分（对坐标）

$$\int_L P\,dx + Q\,dy$$

参数方程，起点 $t=\alpha$，终点 $t=\beta$：

$$\int_L P\,dx + Q\,dy = \int_\alpha^\beta [P(x(t),y(t))x'(t) + Q(x(t),y(t))y'(t)]\,dt$$

注意：积分限与曲线方向有关，起点对应下限，终点对应上限。

3.3 两类曲线积分的关系

$$\int_L P\,dx + Q\,dy = \int_L (P\cos\alpha + Q\cos\beta)\,ds$$

其中 $(\cos\alpha, \cos\beta)$ 是 $L$ 在点 $(x,y)$ 处与方向一致的单位切向量。

3.4 第一类曲线积分的对称性

曲线 $L$ 关于 $y$ 轴对称，$f$ 关于 $x$ 为奇 $\Rightarrow \int_L f\,ds = 0$。其余类似。

4格林公式

4.1 格林公式

$$\oint_L P\,dx + Q\,dy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dxdy$$
条件：$L$ 为 $D$ 的正向边界（逆时针为正），$P,Q$ 在 $D$ 上有连续的一阶偏导数。
注意："正向"指沿 $L$ 行走时 $D$ 在左侧。对于有"洞"的区域，外边界逆时针、内边界顺时针为正。

4.2 平面路径无关与保守场

设 $P,Q$ 在单连通区域 $D$ 上有连续一阶偏导，以下四条等价：

$$\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x} \;\;\text{（在 } D \text{ 内）}$$ $$\Leftrightarrow \oint_L P\,dx+Q\,dy = 0 \;\;\text{（对 } D \text{ 内任意闭曲线）}$$ $$\Leftrightarrow \int_L P\,dx+Q\,dy \text{ 与路径无关（在 } D \text{ 内）}$$ $$\Leftrightarrow P\,dx+Q\,dy \text{ 是某函数 } u(x,y) \text{ 的全微分：} du = P\,dx + Q\,dy$$

4.3 二元原函数的求法

若 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$，则存在 $u(x,y)$ 使 $du = P\,dx + Q\,dy$，且

$$u(x,y) = \int_{x_0}^x P(x,y_0)\,dx + \int_{y_0}^y Q(x,y)\,dy + C$$

或用"凑微分"法直接观察。

4.4 利用格林公式简化计算

技巧：
  • 若 $L$ 不是闭曲线，可补一条线段 $L'$ 使 $L+L'$ 为闭曲线，然后用格林公式，最后减去 $\int_{L'}$
  • 若 $\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}$ 比 $P,Q$ 本身简单，用格林公式更高效
  • 注意奇点：若 $P,Q$ 在 $D$ 内有奇点（如原点），需挖洞处理

5曲面积分

5.1 第一类曲面积分（对面积）

$$\iint_\Sigma f(x,y,z)\,dS$$

若 $\Sigma$ 的方程为 $z = z(x,y)$，投影到 $xy$ 面得 $D_{xy}$：

$$\iint_\Sigma f\,dS = \iint_{D_{xy}} f(x,y,z(x,y))\sqrt{1+z_x^2+z_y^2}\,dxdy$$

5.2 第二类曲面积分（对坐标）

$$\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy$$

若 $\Sigma$ 方程为 $z=z(x,y)$，上侧为正：

$$\iint_\Sigma R\,dxdy = \pm\iint_{D_{xy}} R(x,y,z(x,y))\,dxdy$$

上侧取 $+$，下侧取 $-$。类似地有对 $dydz$ 和 $dzdx$ 的投影公式。

5.3 两类曲面积分的关系

$$\iint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iint_\Sigma (P\cos\alpha + Q\cos\beta + R\cos\gamma)\,dS$$

其中 $(\cos\alpha, \cos\beta, \cos\gamma)$ 是 $\Sigma$ 在指定侧的单位法向量。

6高斯公式

6.1 高斯公式（散度定理）

$$\oiint_\Sigma P\,dydz + Q\,dzdx + R\,dxdy = \iiint_\Omega \left(\frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}\right)dV$$
条件：$\Sigma$ 是 $\Omega$ 的边界曲面取外侧，$P,Q,R$ 在 $\Omega$ 上有连续一阶偏导。

6.2 散度

$$\text{div}\,\vec{F} = \frac{\partial P}{\partial x} + \frac{\partial Q}{\partial y} + \frac{\partial R}{\partial z}$$ $$\oiint_\Sigma \vec{F}\cdot d\vec{S} = \iiint_\Omega \text{div}\,\vec{F}\,dV$$

高斯公式说明：向量场穿过闭曲面的通量等于其散度在体内的体积分。

6.3 使用高斯公式的注意事项

检查清单：
  • 曲面必须封闭。若不封闭，补面后用高斯公式再减去补面的积分
  • 法向量取外侧
  • $P,Q,R$ 在 $\Omega$ 内无奇点。若有奇点（如原点），需挖一个小球面处理
  • 区分 $dydz, dzdx, dxdy$ 分别对应 $P, Q, R$

7斯托克斯公式

7.1 斯托克斯公式

$$\oint_\Gamma P\,dx + Q\,dy + R\,dz = \iint_\Sigma \begin{vmatrix} dydz & dzdx & dxdy \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$ $$= \iint_\Sigma \left(\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z}\right)dydz + \left(\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x}\right)dzdx + \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right)dxdy$$
条件：$\Gamma$ 为空间闭曲线，$\Sigma$ 是以 $\Gamma$ 为边界的曲面，$\Gamma$ 的方向与 $\Sigma$ 的侧符合右手螺旋法则。

7.2 旋度

$$\text{rot}\,\vec{F} = \nabla \times \vec{F} = \begin{vmatrix} \vec{i} & \vec{j} & \vec{k} \\ \frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\ P & Q & R \end{vmatrix}$$ $$\oint_\Gamma \vec{F}\cdot d\vec{r} = \iint_\Sigma (\text{rot}\,\vec{F})\cdot d\vec{S}$$

7.3 三个公式的统一视角

记忆框架：
  • 格林公式：平面闭曲线积分 $\leftrightarrow$ 二重积分（二维斯托克斯）
  • 高斯公式：闭曲面积分 $\leftrightarrow$ 三重积分（散度定理）
  • 斯托克斯公式：空间闭曲线积分 $\leftrightarrow$ 曲面积分（旋度定理）
核心思想都是："边界上的积分 = 内部微分形式的积分"。

8场论初步

8.1 梯度、散度、旋度对比

算子作用对象结果类型公式
梯度 $\nabla f$标量场 $f$向量场$(f_x, f_y, f_z)$
散度 $\nabla\cdot\vec{F}$向量场 $\vec{F}$标量场$P_x+Q_y+R_z$
旋度 $\nabla\times\vec{F}$向量场 $\vec{F}$向量场行列式形式

8.2 保守场（无旋场）

$$\text{rot}\,\vec{F} = \vec{0} \;\Leftrightarrow\; \vec{F} \text{ 为保守场} \;\Leftrightarrow\; \exists\, u, \;\vec{F} = \nabla u$$

保守场中曲线积分与路径无关，沿闭曲线积分为零。

8.3 有势场与势函数

若 $\vec{F} = \nabla u$，则 $u$ 称为 $\vec{F}$ 的势函数。求势函数的方法与求原函数相同（沿路径积分）。
