1不定积分

1.1 原函数与不定积分

若 $F'(x) = f(x)$，则 $F(x)$ 是 $f(x)$ 的一个原函数。$f(x)$ 的全体原函数记为

$$\int f(x)\,dx = F(x) + C$$

其中 $C$ 为任意常数。原函数存在定理：连续函数必有原函数。

1.2 不定积分的性质

$$\frac{d}{dx}\int f(x)\,dx = f(x) \qquad \int F'(x)\,dx = F(x)+C$$ $$\int [k_1 f(x) + k_2 g(x)]\,dx = k_1\!\int f(x)\,dx + k_2\!\int g(x)\,dx$$ $$\int f'(x)\,dx = f(x) + C$$

2换元积分法

2.1 第一换元法（凑微分）

$$\int f[\varphi(x)]\varphi'(x)\,dx = \int f(u)\,du\Big|_{u=\varphi(x)} = F[\varphi(x)] + C$$
关键：识别 $\varphi'(x)\,dx = d[\varphi(x)]$ 的结构。常见凑微分如 $\frac{f'}{f}dx = d\ln|f|$，$f f' dx = \frac{1}{2}d(f^2)$。

2.2 第二换元法

含 $\sqrt{a^2+x^2}$$x = a\tan t$$t = \arctan\frac{x}{a}$
含 $\sqrt{x^2-a^2}$$x = a\sec t$$t = \text{arcsec}\,\frac{x}{a}$
含 $\sqrt[n]{ax+b}$$t = \sqrt[n]{ax+b}$—

2.3 三角代换的辅助三角形

技巧：做三角代换后回代时，画一个直角三角形，标出边的关系，便可轻松读出 $\sin t$、$\cos t$、$\tan t$ 的值。

3分部积分法

3.1 公式

$$\int u\,dv = uv - \int v\,du$$

3.2 选 $u$ 的优先顺序："反对幂三指"

优先级函数类型取 $u$$dv$ 取
1反三角函数$\arctan x$其余
2对数函数$\ln x$其余
3幂函数$x^n$其余
4三角函数$\sin x$其余
5指数函数—（取 $dv = e^x dx$）$e^x dx$
口诀："反对幂三指"——排在前面的是 $u$，排在后面的进 $dv$。

3.3 递推型分部积分

$$I_n = \int \sin^n x\,dx = -\frac{\sin^{n-1}x\cos x}{n} + \frac{n-1}{n}I_{n-2} \quad (n \geq 2)$$ $$I_n = \int \frac{dx}{(x^2+a^2)^n} = \frac{x}{2a^2(n-1)(x^2+a^2)^{n-1}} + \frac{2n-3}{2a^2(n-1)}I_{n-1} \quad (n \geq 2)$$

3.4 "回归型"分部积分

形如 $\int e^{ax}\sin bx\,dx$ 或 $\int e^{ax}\cos bx\,dx$，分部两次后出现原式，解方程：

$$\int e^x \sin x\,dx = \frac{e^x(\sin x - \cos x)}{2} + C \qquad \int e^x \cos x\,dx = \frac{e^x(\sin x + \cos x)}{2} + C$$

3.5 有理函数积分

将有理分式 $\frac{P(x)}{Q(x)}$ 分解为部分分式后逐项积分：

$$\frac{1}{(x-a)(x-b)} = \frac{1}{b-a}\left(\frac{1}{x-a}-\frac{1}{x-b}\right) \quad (a \neq b)$$ $$\frac{1}{(x-a)^2+b^2} \;\to\; \text{令 } x-a = b\tan t$$

4定积分

4.1 定积分定义

$$\int_a^b f(x)\,dx = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i)\Delta x_i$$

当 $f \in C[a,b]$ 时，该极限存在（即可积），且与分割方式及 $\xi_i$ 取法无关。

4.2 可积条件

  • $f \in C[a,b] \Rightarrow$ 可积
  • $f$ 在 $[a,b]$ 上有界且只有有限个间断点 $\Rightarrow$ 可积
  • $f$ 在 $[a,b]$ 上单调 $\Rightarrow$ 可积

4.3 牛顿—莱布尼茨公式

$$\int_a^b f(x)\,dx = F(b) - F(a) = \Big[F(x)\Big]_a^b, \quad F'(x) = f(x)$$
意义：将定积分（极限求和）与原函数（反导数）联系起来，是微积分学最核心的定理。

4.4 定积分的性质

$$\int_a^a f\,dx = 0 \qquad \int_a^b f\,dx = -\int_b^a f\,dx$$ $$\int_a^b [k_1 f + k_2 g]\,dx = k_1\!\int_a^b f\,dx + k_2\!\int_a^b g\,dx$$ $$\int_a^b f\,dx = \int_a^c f\,dx + \int_c^b f\,dx$$ $$f \leq g \;\Rightarrow\; \int_a^b f\,dx \leq \int_a^b g\,dx$$ $$\left|\int_a^b f\,dx\right| \leq \int_a^b |f|\,dx$$

4.5 积分中值定理

$$f \in C[a,b] \;\Rightarrow\; \exists\, \xi \in [a,b], \;\; \int_a^b f(x)\,dx = f(\xi)(b-a)$$

推广（加权积分中值定理）：若 $g(x)$ 不变号，则 $\exists\,\xi \in[a,b]$，使 $\int_a^b f\cdot g\,dx = f(\xi)\int_a^b g\,dx$。

4.6 定积分的对称性

$$\int_{-a}^a f(x)\,dx = \begin{cases} 0, & f \text{ 为奇函数} \\ 2\int_0^a f(x)\,dx, & f \text{ 为偶函数} \end{cases}$$ $$f(x+T)=f(x) \;\Rightarrow\; \int_a^{a+T} f\,dx = \int_0^T f\,dx \quad (\forall a)$$

4.7 有用的定积分结果

$$\int_0^{\pi/2} \sin^n x\,dx = \int_0^{\pi/2} \cos^n x\,dx = \frac{(n-1)!!}{n!!}\times\begin{cases}\pi/2 & n\text{ 偶}\\ 1 & n\text{ 奇}\end{cases}$$ $$\int_0^\pi x\,f(\sin x)\,dx = \frac{\pi}{2}\int_0^\pi f(\sin x)\,dx$$ $$\int_0^{\pi/2} f(\sin x)\,dx = \int_0^{\pi/2} f(\cos x)\,dx$$

5变限积分与微积分基本定理

5.1 变上限积分求导

$$\frac{d}{dx}\int_a^x f(t)\,dt = f(x) \quad (f \text{ 连续})$$ $$\frac{d}{dx}\int_{\varphi(x)}^{\psi(x)} f(t)\,dt = f(\psi(x))\,\psi'(x) - f(\varphi(x))\,\varphi'(x)$$
注意：若被积函数含积分变量 $x$，需先分离或换元。例如 $\int_a^x xf(t)\,dt = x\int_a^x f(t)\,dt$，用乘法求导法则。

5.2 原函数存在定理

若 $f \in C[a,b]$，则 $\Phi(x) = \int_a^x f(t)\,dt$ 是 $f(x)$ 的一个原函数。

6反常积分

6.1 无穷限反常积分

$$\int_a^{+\infty} f\,dx = \lim_{b \to +\infty}\int_a^b f\,dx \qquad \int_{-\infty}^b f\,dx = \lim_{a \to -\infty}\int_a^b f\,dx$$ $$\int_{-\infty}^{+\infty} f\,dx = \int_{-\infty}^c f\,dx + \int_c^{+\infty} f\,dx$$

6.2 无界函数的反常积分（瑕积分）

若 $f$ 在 $x_0$ 附近无界（$x_0$ 为瑕点）：

$$\int_a^{x_0} f\,dx = \lim_{\varepsilon \to 0^+}\int_a^{x_0-\varepsilon} f\,dx$$

6.3 敛散性判据（$p$-积分）

$$\int_a^{+\infty} \frac{dx}{x^p} \;\text{收敛} \Leftrightarrow p > 1 \quad (a > 0)$$ $$\int_0^a \frac{dx}{x^p} \;\text{收敛} \Leftrightarrow p < 1 \quad (a > 0, \text{瑕点 } x=0)$$

6.4 比较判敛法

设 $f(x) \geq g(x) \geq 0$：

  • 若 $\int f$ 收敛 $\Rightarrow$ $\int g$ 收敛
  • 若 $\int g$ 发散 $\Rightarrow$ $\int f$ 发散
  • 比较极限：$\lim \frac{f}{g} = c$，$c>0 \Rightarrow$ 同敛散；$c=0$ 且 $\int g$ 收敛 $\Rightarrow$ $\int f$ 收敛

7定积分的几何应用

7.1 平面图形面积

坐标系面积公式
直角坐标（$x$ 型区域）$A = \int_a^b |f(x)-g(x)|\,dx$
直角坐标（$y$ 型区域）$A = \int_c^d |x_2(y)-x_1(y)|\,dy$
极坐标$A = \frac{1}{2}\int_\alpha^\beta [r_2(\theta)^2 - r_1(\theta)^2]\,d\theta$

7.2 旋转体体积

方法公式适用场景
圆盘法（绕 $x$ 轴）$V = \pi\int_a^b [f(x)]^2\,dx$$y=f(x)$ 绕 $x$ 轴
圆盘法（绕 $y$ 轴）$V = \pi\int_c^d [\varphi(y)]^2\,dy$$x=\varphi(y)$ 绕 $y$ 轴
柱壳法（绕 $y$ 轴）$V = 2\pi\int_a^b x\,|f(x)|\,dx$$y=f(x)$ 绕 $y$ 轴
washer 法$V = \pi\int_a^b [f(x)^2 - g(x)^2]\,dx$两曲线之间绕 $x$ 轴

7.3 平行截面面积已知的立体体积

$$V = \int_a^b A(x)\,dx$$

其中 $A(x)$ 为在 $x$ 处的截面面积。

7.4 弧长

曲线方程形式弧长公式
$y = f(x), \; a \leq x \leq b$$s = \int_a^b \sqrt{1+[f'(x)]^2}\,dx$
参数方程 $x=\varphi(t), y=\psi(t)$$s = \int_\alpha^\beta \sqrt{[\varphi'(t)]^2+[\psi'(t)]^2}\,dt$
极坐标 $r = r(\theta)$$s = \int_\alpha^\beta \sqrt{r^2+[r'(\theta)]^2}\,d\theta$

7.5 旋转面的面积

$$S = 2\pi\int_a^b |y|\sqrt{1+y'^2}\,dx \quad (\text{绕 } x \text{ 轴})$$ $$S = 2\pi\int_\alpha^\beta r(\theta)\sin\theta\sqrt{r^2+r'^2}\,d\theta \quad (\text{极坐标，绕极轴})$$

8定积分的物理应用

物理量公式
变力做功$W = \int_a^b F(x)\,dx$
液体静压力$P = \rho g \int_a^b x\,f(x)\,dx$（$x$ 为深度）
细棒引力/电场用微元法 $dF = k\frac{dm}{r^2}$
质心坐标（$x$ 轴）$\bar{x} = \frac{\int_a^b x\,f(x)\,dx}{\int_a^b f(x)\,dx}$
函数平均值$\bar{f} = \frac{1}{b-a}\int_a^b f(x)\,dx$

8.1 微元法（元素法）核心步骤

  1. 选取积分变量 $x$，确定范围 $[a,b]$
  2. 取微元 $[x, x+dx]$，求出局部量 $dU = f(x)\,dx$
  3. 积分 $U = \int_a^b f(x)\,dx$
关键：微元 $dU$ 必须是 $dx$ 的线性主部，即 $dU = f(x)dx + o(dx)$。
