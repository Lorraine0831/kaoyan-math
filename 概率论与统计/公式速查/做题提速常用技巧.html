2标准化技巧

2.1 正态分布标准化

$$X \sim N(\mu, \sigma^2) \;\Rightarrow\; Z = \frac{X - \mu}{\sigma} \sim N(0, 1)$$ $$P(X < x) = \Phi\!\left(\frac{x-\mu}{\sigma}\right)$$
秒杀：任何正态分布的概率计算，先标准化为 $N(0,1)$，然后查标准正态表。

2.2 标准正态的分位数

$$\Phi(1.64) \approx 0.95 \;\Rightarrow\; z_{0.05} \approx 1.64 \;\text{（单侧）}$$ $$\Phi(1.96) \approx 0.975 \;\Rightarrow\; z_{0.025} \approx 1.96 \;\text{（双侧}\alpha=0.05\text{）}$$ $$\Phi(2.58) \approx 0.995 \;\Rightarrow\; z_{0.005} \approx 2.58 \;\text{（双侧}\alpha=0.01\text{）}$$ $$\Phi(-x) = 1 - \Phi(x) \qquad \Phi(0) = 0.5$$

2.3 中心极限定理标准化

$$\sum_{i=1}^n X_i \;\dot{\sim}\; N(n\mu, \; n\sigma^2) \;\Rightarrow\; P\!\left(\sum X_i \leq x\right) \approx \Phi\!\left(\frac{x - n\mu}{\sigma\sqrt{n}}\right)$$ $$\bar{X} \;\dot{\sim}\; N\!\left(\mu, \;\frac{\sigma^2}{n}\right) \;\Rightarrow\; P\!\left(\bar{X} \leq x\right) \approx \Phi\!\left(\frac{x - \mu}{\sigma/\sqrt{n}}\right)$$

3对称性利用

3.1 正态分布的对称性

$$X \sim N(0,1) \;\Rightarrow\; P(X > 0) = P(X < 0) = 0.5$$ $$P(|X| < a) = 2\Phi(a) - 1 \qquad P(X > a) = 1 - \Phi(a) = \Phi(-a)$$ $$P(X < -a) = P(X > a) \;\text{（关于 } \mu \text{ 对称）}$$

3.2 $t$ 分布的对称性

$$T \sim t(n) \;\Rightarrow\; \text{密度关于 } 0 \text{ 对称}, \;\; P(T > t_\alpha) = P(T < -t_\alpha)$$ $$t_{1-\alpha}(n) = -t_\alpha(n)$$

3.3 指数分布的无记忆性

$$X \sim Exp(\lambda) \;\Rightarrow\; P(X > s + t \;|\; X > s) = P(X > t)$$
直觉：指数分布"忘了"已经等了多久——已存活 $s$ 时间后再存活 $t$ 时间的概率，和从头开始一样。这是指数分布的特征性质。

4期望与方差的快速推导

4.1 方差计算公式

$$D(X) = E(X^2) - [E(X)]^2$$
考试最常用：不要用定义 $E[(X-EX)^2]$ 去算，直接用 $E(X^2) - (EX)^2$，分两步算更简洁。

4.2 利用线性性质快速求期望

$$E(aX + b) = aE(X) + b \qquad E(aX + bY) = aE(X) + bE(Y)$$ $$D(aX + b) = a^2 D(X) \qquad D(aX + bY) = a^2 D(X) + b^2 D(Y) + 2ab\,\text{Cov}(X,Y)$$

4.3 独立时方差可加

$$X, Y \text{ 独立} \;\Rightarrow\; D(X \pm Y) = D(X) + D(Y)$$ $$X, Y \text{ 不相关} \;\Rightarrow\; \text{Cov}(X,Y) = 0 \;\Rightarrow\; D(X \pm Y) = D(X) + D(Y)$$

4.4 协方差的快速计算

$$\text{Cov}(X,Y) = E(XY) - E(X)E(Y)$$ $$\text{Cov}(aX+b, \, cY+d) = ac\,\text{Cov}(X,Y)$$ $$\text{Cov}(X+Y, \, Z) = \text{Cov}(X,Z) + \text{Cov}(Y,Z)$$

5常见做题技巧

5.1 分布函数法求密度

$$F_Y(y) = P(Y \leq y) = P(g(X) \leq y) \;\;\text{解出 } X \text{ 的范围，用 } F_X \text{ 表示}$$ $$f_Y(y) = F_Y'(y)$$
步骤：先求 $Y = g(X)$ 的分布函数 $F_Y(y)$，再求导得密度 $f_Y(y)$。关键是把 $P(g(X) \leq y)$ 转化为 $P(X \leq h(y))$。

5.2 卷积公式（$Z = X+Y$）

$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z - x)\,dx$$

使用要点：

  • 仅当 $X, Y$ 独立时，$f(x,y) = f_X(x)f_Y(y)$，才能用卷积公式
  • 积分限由密度函数的定义域确定（如均匀分布 $[0,1]$ 则 $0 \leq x \leq 1$ 且 $0 \leq z-x \leq 1$）

5.3 最大值/最小值分布

$$M = \max(X_1, \ldots, X_n), \;\; X_i \text{ 独立}$$ $$F_M(z) = F_{X_1}(z) \cdot F_{X_2}(z) \cdots F_{X_n}(z) = \prod F_{X_i}(z)$$ $$N = \min(X_1, \ldots, X_n)$$ $$F_N(z) = 1 - \prod [1 - F_{X_i}(z)]$$
记忆：最大值"都 $\leq$" → 分布函数相乘；最小值"至少有一个 $\leq$" → 用对立事件"都 $>$" → $1 - \prod(1-F)$。

5.4 二项分布的分解

$$X = X_1 + X_2 + \cdots + X_n, \;\; X_i \sim B(1,p) \;\text{独立}$$ $$\Rightarrow X \sim B(n,p), \;\; E(X) = np, \;\; D(X) = np(1-p)$$

5.5 正态分布的可加性

$$X \sim N(\mu_1, \sigma_1^2), \; Y \sim N(\mu_2, \sigma_2^2), \; X, Y \text{ 独立}$$ $$\Rightarrow aX + bY \sim N(a\mu_1 + b\mu_2, \; a^2\sigma_1^2 + b^2\sigma_2^2)$$
注意：必须独立才能用。不独立时方差需加 $2ab\,\text{Cov}(X,Y)$ 项。

6抽样分布速记

6.1 构造规则

分布构造方法自由度关键性质
$\chi^2$$\sum Z_i^2$, $Z_i\sim N(0,1)$ 独立$n$$E=n$, $D=2n$, 可加性
$t$$\frac{Z}{\sqrt{\chi^2/n}}$, $Z$ 与 $\chi^2$ 独立$n$关于0对称, $n\to\infty \to N(0,1)$
$F$$\frac{\chi_1^2/m}{\chi_2^2/n}$, 两个 $\chi^2$ 独立$(m,n)$$1/F \sim F(n,m)$

6.2 正态总体的"四大抽样分布"

$$\frac{\bar{X}-\mu}{\sigma/\sqrt{n}} \sim N(0,1) \qquad \frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$ $$\frac{\bar{X}-\mu}{S/\sqrt{n}} \sim t(n-1) \qquad \frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1-1, n_2-1)$$
记忆线索：用 $\sigma$ 已知→$Z$ 或 $\chi^2$；用 $S$（样本标准差）→$t$ 或 $F$。$\sigma$ 未知时用 $S$ 替代，分布从 $Z$ 变 $t$、从 $\chi^2$ 不变但自由度减1。

6.3 $\chi^2$ 的可加性

$$\chi_1^2 \sim \chi^2(n_1), \;\; \chi_2^2 \sim \chi^2(n_2), \;\; \text{独立} \;\Rightarrow\; \chi_1^2 + \chi_2^2 \sim \chi^2(n_1 + n_2)$$

7公式速记卡

7.1 全概率与贝叶斯的核心思路

全概率："由因推果"——已知各"因"的概率及各"因"下结果的概率，求结果的概率。
贝叶斯："由果推因"——已知结果发生了，反推各"因"的概率。$P(\text{因}|\text{果}) = \frac{P(\text{因})P(\text{果}|\text{因})}{P(\text{果})}$。

7.2 估计方法对照

方法思路步骤
矩估计用样本矩替代总体矩令 $E(X)=\bar{X}$, $E(X^2)=\frac{1}{n}\sum X_i^2$，解出参数
最大似然使样本出现的概率最大构造 $L=\prod f(x_i;\theta)$ → $\ln L$ → 求导=0

7.3 常用等价关系

$$X, Y \text{ 独立} \;\Rightarrow\; \text{Cov}(X,Y) = 0 \;\Leftrightarrow\; E(XY) = E(X)E(Y) \;\Leftrightarrow\; \rho = 0$$ $$\text{但} \;\rho=0 \;\nRightarrow\; \text{独立（除非正态）}$$ $$\text{二维正态：} \;\rho = 0 \;\Leftrightarrow\; \text{独立}$$
