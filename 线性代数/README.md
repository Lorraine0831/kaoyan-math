# 线性代数复习资料

考前系统复习专用，涵盖知识点、必背公式、做题提速技巧。

## 目录结构

```
线性代数/
├── index.html                  ← 导航首页
├── README.md
├── assets/
│   └── style.css
├── 知识点/
│   ├── 01_行列式.html
│   ├── 02_矩阵.html
│   ├── 03_向量与方程组.html
│   └── 04_特征值与二次型.html
└── 公式速查/
    ├── 必背公式.html
    └── 做题提速常用技巧.html
```

## 使用方式

1. 浏览器打开 `index.html` 进入导航首页
2. 所有页面支持 MathJax 公式渲染（需联网加载）
3. 支持暗色模式（跟随系统）
