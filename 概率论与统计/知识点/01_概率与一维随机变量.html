1概率的基本概念

1.1 样本空间与事件

  • 样本空间 $\Omega$：所有可能结果的集合
  • 事件：样本空间的子集
  • 必然事件 $\Omega$，不可能事件 $\varnothing$

1.2 概率公理（柯尔莫哥洛夫）

$$\textbf{非负性：} \;\; P(A) \geq 0$$ $$\textbf{规范性：} \;\; P(\Omega) = 1$$ $$\textbf{可列可加性：} \;\; A_i A_j = \varnothing \;(i \neq j) \;\Rightarrow\; P\!\left(\bigcup A_i\right) = \sum P(A_i)$$

1.3 加法公式

$$P(A \cup B) = P(A) + P(B) - P(AB)$$ $$P(AB) = P(A) + P(B) - P(A \cup B)$$ $$P(\bar{A}) = 1 - P(A) \qquad P(\bar{A}B) = P(B) - P(AB)$$

2条件概率与独立性

2.1 条件概率

$$P(A|B) = \frac{P(AB)}{P(B)} \quad (P(B) > 0)$$
理解：条件概率 $P(A|B)$ 是在"已知 $B$ 发生"的条件下 $A$ 发生的概率，此时样本空间从 $\Omega$ 缩小为 $B$。

2.2 乘法公式

$$P(AB) = P(B) \cdot P(A|B) = P(A) \cdot P(B|A)$$ $$P(ABC) = P(A) \cdot P(B|A) \cdot P(C|AB)$$

2.3 全概率公式

设 $\{A_1, \ldots, A_n\}$ 为完备事件组（互斥且并集为 $\Omega$），则

$$P(B) = \sum_{i=1}^n P(A_i) P(B|A_i)$$
思路："由因推果"——$B$ 的发生由各"因" $A_i$ 引起，按权重 $P(A_i)$ 加权平均。

2.4 贝叶斯公式

$$P(A_k|B) = \frac{P(A_k)P(B|A_k)}{\sum_{i=1}^n P(A_i)P(B|A_i)}$$
思路："由果推因"——已知 $B$ 发生了，反推各"因" $A_k$ 的概率。分子是"该因导致果"的概率，分母是全概率。

2.5 事件的独立性

$$A, B \text{ 独立} \;\Leftrightarrow\; P(AB) = P(A)P(B) \;\Leftrightarrow\; P(A|B) = P(A)$$ $$A, B \text{ 独立} \;\Rightarrow\; A, \bar{B}; \;\; \bar{A}, B; \;\; \bar{A}, \bar{B} \text{ 都独立}$$

2.6 伯努利概型

$n$ 次独立重复试验，每次成功概率 $p$：

$$P(\text{恰好 } k \text{ 次成功}) = \binom{n}{k} p^k (1-p)^{n-k}$$

3一维离散型随机变量

3.1 分布律

$$P(X = x_k) = p_k, \quad k = 1, 2, \ldots \qquad p_k \geq 0, \;\; \sum p_k = 1$$

3.2 分布函数

$$F(x) = P(X \leq x) = \sum_{x_k \leq x} p_k$$

分布函数是右连续的阶梯函数，在 $x_k$ 处有跃度 $p_k$。

3.3 常见离散分布

分布分布律$E(X)$$D(X)$
0-1分布$P(X=k) = p^k(1-p)^{1-k}$, $k=0,1$$p$$p(1-p)$
二项分布 $B(n,p)$$P(X=k) = \binom{n}{k}p^k(1-p)^{n-k}$$np$$np(1-p)$
泊松分布 $\pi(\lambda)$$P(X=k) = \frac{\lambda^k}{k!}e^{-\lambda}$$\lambda$$\lambda$
几何分布$P(X=k) = (1-p)^{k-1}p$, $k=1,2,\ldots$$\frac{1}{p}$$\frac{1-p}{p^2}$

3.4 泊松定理

$$\binom{n}{k}p^k(1-p)^{n-k} \approx \frac{\lambda^k}{k!}e^{-\lambda}, \quad \lambda = np, \;\; n \to \infty, \; p \to 0$$

4一维连续型随机变量

4.1 概率密度函数

$$f(x) \geq 0, \quad \int_{-\infty}^{+\infty} f(x)\,dx = 1$$ $$F(x) = \int_{-\infty}^x f(t)\,dt \qquad f(x) = F'(x)$$ $$P(a < X < b) = \int_a^b f(x)\,dx = F(b) - F(a)$$ $$P(X = a) = 0 \;\;\text{（连续型在单点概率为零）}$$

4.2 常见连续分布

分布密度函数$E(X)$$D(X)$
均匀 $U(a,b)$$f(x) = \frac{1}{b-a}$, $a \leq x \leq b$$\frac{a+b}{2}$$\frac{(b-a)^2}{12}$
指数 $Exp(\lambda)$$f(x) = \lambda e^{-\lambda x}$, $x > 0$$\frac{1}{\lambda}$$\frac{1}{\lambda^2}$
正态 $N(\mu, \sigma^2)$$f(x) = \frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$\mu$$\sigma^2$

4.3 指数分布的无记忆性

$$P(X > s+t \;|\; X > s) = P(X > t)$$

4.4 正态分布的性质

$$X \sim N(\mu, \sigma^2) \;\Rightarrow\; Y = aX + b \sim N(a\mu + b, \; a^2\sigma^2)$$ $$\text{标准化：} \;\; Z = \frac{X - \mu}{\sigma} \sim N(0,1)$$ $$P(X \leq x) = \Phi\!\left(\frac{x - \mu}{\sigma}\right) \qquad \Phi(-x) = 1 - \Phi(x)$$ $$\text{3$\sigma$ 法则：} \;\; P(|X - \mu| < 3\sigma) \approx 0.9974$$

5随机变量函数的分布

5.1 离散型

若 $Y = g(X)$，则 $P(Y = y_j) = \sum_{g(x_i) = y_j} P(X = x_i)$。

5.2 连续型：分布函数法

$$F_Y(y) = P(Y \leq y) = P(g(X) \leq y)$$ $$\text{解出 } X \text{ 的范围，用 } F_X \text{ 表示}$$ $$f_Y(y) = F_Y'(y)$$

5.3 公式法（单调函数）

若 $y = g(x)$ 严格单调且可导，反函数 $x = h(y)$，则

$$f_Y(y) = f_X(h(y)) \cdot |h'(y)|$$
注意：公式法只适用于 $g$ 严格单调的情况。若不单调（如 $Y = X^2$），必须用分布函数法分段处理。
