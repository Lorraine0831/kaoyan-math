1概率基本公式

1.1 加法公式

$$P(A \cup B) = P(A) + P(B) - P(AB)$$ $$P(A \cup B \cup C) = P(A)+P(B)+P(C) - P(AB)-P(AC)-P(BC) + P(ABC)$$ $$P(\bar{A}) = 1 - P(A)$$

1.2 条件概率与乘法公式

$$P(A|B) = \frac{P(AB)}{P(B)} \quad (P(B)>0)$$ $$P(AB) = P(B)\cdot P(A|B) \quad \text{（乘法公式）}$$ $$P(ABC) = P(A)\cdot P(B|A)\cdot P(C|AB)$$

1.3 全概率公式与贝叶斯公式

$$\text{全概率：} \;\; P(B) = \sum_{i=1}^n P(A_i)P(B|A_i) \quad \text{（} \{A_i\} \text{ 为完备事件组）}$$ $$\text{贝叶斯：} \;\; P(A_k|B) = \frac{P(A_k)P(B|A_k)}{\sum_{i=1}^n P(A_i)P(B|A_i)}$$

1.4 独立性

$$A, B \text{ 独立} \;\Leftrightarrow\; P(AB) = P(A)P(B)$$ $$A, B \text{ 独立} \;\Leftrightarrow\; A, \bar{B} \text{ 独立} \;\Leftrightarrow\; \bar{A}, \bar{B} \text{ 独立}$$ $$A_1, \ldots, A_n \text{ 相互独立} \;\Rightarrow\; \text{其中任意 } k \text{ 个也独立}$$

1.5 伯努利概型

$$P(\text{恰好 } k \text{ 次成功}) = \binom{n}{k}p^k(1-p)^{n-k} \quad (\text{二项分布})$$

2.3 正态分布的重要性质

$$X \sim N(\mu, \sigma^2) \;\Rightarrow\; Y = aX+b \sim N(a\mu+b, \; a^2\sigma^2)$$ $$X \sim N(0,1) \;\Rightarrow\; \Phi(-x) = 1 - \Phi(x) \qquad \Phi(0) = 0.5$$ $$P(|X-\mu|<\sigma) \approx 0.6826, \;\; P(|X-\mu|<2\sigma) \approx 0.9544, \;\; P(|X-\mu|<3\sigma) \approx 0.9974$$ $$X \sim N(\mu_1, \sigma_1^2), \; Y \sim N(\mu_2, \sigma_2^2), \; \text{独立} \;\Rightarrow\; X \pm Y \sim N(\mu_1 \pm \mu_2, \; \sigma_1^2 + \sigma_2^2)$$

2.4 泊松定理（二项分布的泊松逼近）

$$\binom{n}{k}p^k(1-p)^{n-k} \approx \frac{\lambda^k}{k!}e^{-\lambda}, \quad \lambda = np, \; n \to \infty, \; p \to 0$$

3期望、方差与矩

3.1 期望定义与计算

$$\text{离散：} \;\; E(X) = \sum_i x_i p_i \qquad \text{连续：} \;\; E(X) = \int_{-\infty}^{+\infty} x\,f(x)\,dx$$ $$E(g(X)) = \sum_i g(x_i)p_i \quad\text{或}\quad E(g(X)) = \int g(x)f(x)\,dx$$ $$E(g(X,Y)) = \iint g(x,y)f(x,y)\,dxdy$$

3.2 期望的性质

$$E(C) = C \qquad E(CX) = CE(X) \qquad E(X+C) = E(X)+C$$ $$E(X \pm Y) = E(X) \pm E(Y) \qquad E(XY) = E(X)E(Y) \;\text{（}X,Y\text{ 独立时）}$$ $$E\!\left(\sum a_i X_i\right) = \sum a_i E(X_i)$$

3.3 方差定义与性质

$$D(X) = E[(X-EX)^2] = E(X^2) - [E(X)]^2$$ $$D(C) = 0 \qquad D(CX) = C^2 D(X) \qquad D(X+C) = D(X)$$ $$D(X \pm Y) = D(X) + D(Y) \pm 2\,\text{Cov}(X,Y)$$ $$D\!\left(\sum a_i X_i\right) = \sum a_i^2 D(X_i) + 2\sum_{i

3.4 协方差与相关系数

$$\text{Cov}(X,Y) = E[(X-EX)(Y-EY)] = E(XY) - E(X)E(Y)$$ $$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)}\sqrt{D(Y)}} \in [-1, 1]$$ $$|\rho| = 1 \;\Leftrightarrow\; P(Y = aX+b) = 1$$ $$\rho = 0 \;\Leftrightarrow\; \text{不相关} \quad (\text{独立} \;\Rightarrow\; \text{不相关，反之不成立})$$ $$\text{Cov}(X,X) = D(X) \qquad \text{Cov}(aX+b, cY+d) = ac\,\text{Cov}(X,Y)$$

3.5 矩

$$k \text{ 阶原点矩：} \; E(X^k) \qquad k \text{ 阶中心矩：} \; E[(X-EX)^k]$$ $$\text{偏度} = \frac{E[(X-EX)^3]}{[D(X)]^{3/2}} \qquad \text{峰度} = \frac{E[(X-EX)^4]}{[D(X)]^2}$$

4二维随机变量

4.1 联合分布函数

$$F(x,y) = P(X \leq x, Y \leq y)$$ $$f(x,y) \geq 0, \;\; \iint f(x,y)\,dxdy = 1, \;\; F(x,y) = \int_{-\infty}^y\int_{-\infty}^x f(u,v)\,dudv$$

4.2 边际分布

$$\text{离散：} \;\; p_{i\cdot} = \sum_j p_{ij}, \quad p_{\cdot j} = \sum_i p_{ij}$$ $$\text{连续：} \;\; f_X(x) = \int_{-\infty}^{+\infty} f(x,y)\,dy, \quad f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)\,dx$$

4.3 条件分布

$$\text{离散：} \;\; P(X=x_i | Y=y_j) = \frac{p_{ij}}{p_{\cdot j}}$$ $$\text{连续：} \;\; f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)} \quad (f_Y(y) > 0)$$

4.4 独立性

$$X, Y \text{ 独立} \;\Leftrightarrow\; F(x,y) = F_X(x) \cdot F_Y(y)$$ $$\Leftrightarrow\; p_{ij} = p_{i\cdot} \cdot p_{\cdot j} \;\text{（离散）}$$ $$\Leftrightarrow\; f(x,y) = f_X(x) \cdot f_Y(y) \;\text{（连续）}$$

4.5 二维正态分布

$$(X,Y) \sim N(\mu_1, \mu_2, \sigma_1^2, \sigma_2^2, \rho)$$ $$X \sim N(\mu_1, \sigma_1^2), \; Y \sim N(\mu_2, \sigma_2^2)$$ $$X, Y \text{ 独立} \;\Leftrightarrow\; \rho = 0$$ $$\text{Cov}(X,Y) = \rho\sigma_1\sigma_2$$

4.6 函数的分布

$Z = X + Y$ 的密度（卷积公式）：

$$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x) f_Y(z-x)\,dx = \int_{-\infty}^{+\infty} f(x, z-x)\,dx$$

$Z = X + Y$ 离散情形：

$$P(Z = z_k) = \sum_{i} P(X=x_i) P(Y = z_k - x_i)$$

最大值/最小值分布：

$$F_{\max}(z) = F_X(z) \cdot F_Y(z) \quad (X, Y \text{ 独立})$$ $$F_{\min}(z) = 1 - [1 - F_X(z)][1 - F_Y(z)]$$

5大数定律与中心极限定理

5.1 切比雪夫不等式

$$P(|X - E(X)| \geq \varepsilon) \leq \frac{D(X)}{\varepsilon^2}$$

5.2 大数定律

$$\text{切比雪夫大数定律：} \;\; \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \frac{1}{n}\sum E(X_i)$$ $$\text{辛钦大数定律：} \;\; \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \mu \;\; (X_i \text{ 独立同分布}, E(X_i)=\mu)$$ $$\text{伯努利大数定律：} \;\; \frac{n_A}{n} \xrightarrow{P} p \;\; (n_A \text{ 为 } n \text{ 次独立试验中 } A \text{ 发生次数})$$

5.3 中心极限定理

$$\text{林德伯格-列维：} \;\; X_1, \ldots, X_n \text{ i.i.d.}, \; E(X_i)=\mu, \; D(X_i)=\sigma^2$$ $$\frac{\sum X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1) \;\;\text{即}\;\; \sum X_i \dot{\sim} N(n\mu, \; n\sigma^2)$$ $$\text{棣莫弗-拉普拉斯：} \;\; X \sim B(n,p) \;\Rightarrow\; \frac{X - np}{\sqrt{np(1-p)}} \dot{\sim} N(0,1)$$

6数理统计基础

6.1 三大抽样分布

$$\chi^2 = \sum_{i=1}^n Z_i^2 \sim \chi^2(n), \;\; Z_i \sim N(0,1) \;\text{独立}$$ $$E(\chi^2(n)) = n, \;\; D(\chi^2(n)) = 2n$$ $$T = \frac{Z}{\sqrt{\chi^2/n}} \sim t(n), \;\; E(T) = 0 \;(n>1), \;\; D(T) = \frac{n}{n-2} \;(n>2)$$ $$F = \frac{\chi_1^2/m}{\chi_2^2/n} \sim F(m,n), \;\; \frac{1}{F} \sim F(n,m)$$

6.2 正态总体的抽样分布

$$X_1, \ldots, X_n \sim N(\mu, \sigma^2) \;\text{i.i.d.}, \;\; \bar{X} = \frac{1}{n}\sum X_i, \;\; S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$$ $$\bar{X} \sim N\!\left(\mu, \;\frac{\sigma^2}{n}\right) \;\;\Rightarrow\;\; \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$$ $$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$ $$\frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$$ $$\bar{X} \text{ 与 } S^2 \text{ 独立}$$

6.3 参数估计

矩估计：令 $E(X) = \bar{X}$（一阶），$E(X^2) = \frac{1}{n}\sum X_i^2$（二阶），解出参数。

最大似然估计：构造似然函数 $L(\theta) = \prod f(x_i;\theta)$，取对数 $\ln L$，对 $\theta$ 求导令其为零。

$$\hat{\theta}_{MLE}: \;\; \frac{\partial \ln L(\theta)}{\partial \theta} = 0$$

6.4 区间估计

参数置信区间（$\sigma^2$ 已知）置信区间（$\sigma^2$ 未知）
$\mu$（单正态总体）$\bar{X} \pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}}$$\bar{X} \pm t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}}$
$$\sigma^2 \text{ 的区间：} \;\; \left(\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \;\frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right)$$

6.5 假设检验

$$H_0: \mu = \mu_0 \quad \text{vs} \quad H_1: \mu \neq \mu_0$$ $$\sigma^2 \text{ 已知：} \;\; W = \left\{|Z| > z_{\alpha/2}\right\}, \;\; Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}$$ $$\sigma^2 \text{ 未知：} \;\; W = \left\{|T| > t_{\alpha/2}(n-1)\right\}, \;\; T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}$$

两类错误：

  • 第一类错误（弃真）：$P(\text{拒绝}H_0 | H_0 \text{ 真}) = \alpha$
  • 第二类错误（取伪）：$P(\text{接受}H_0 | H_1 \text{ 真}) = \beta$

7.6 连续型随机变量函数的分布——常见"坑"

三大常见错误：
  • 直接代公式：$Y=g(X)$ 只在 $g$ 严格单调时才能用公式法；若不单调（如 $Y=X^2$），必须用分布函数法分段处理
  • 漏定义域：卷积公式求 $f_Z(z)$ 时，积分限必须同时满足 $f_X(x)$ 和 $f_Y(z-x)$ 的非零区间
  • 错用独立条件：卷积公式 $\int f_X(x)f_Y(z-x)dx$ 只对独立变量成立；不独立时用 $\int f(x,z-x)dx$

7.7 矩估计与最大似然估计的"陷阱"

$$\text{均匀分布 } U(0,\theta) \text{ 的 MLE：} \;\; \hat{\theta} = \max(X_1,\ldots,X_n) \text{（不能用求导法！）}$$
注意：均匀分布 $U(0,\theta)$ 的最大似然估计在边界取到，常规的"求导=0"不适用——似然函数 $\frac{1}{\theta^n}$ 关于 $\theta$ 单调递减，应在满足 $X_{(n)}\leq\theta$ 的前提下取最小的 $\theta$，即 $\hat{\theta} = X_{(n)} = \max_i X_i$。

7.8 正态总体的"一句话"记忆

$$\text{已知 } \sigma^2 \to Z\text{/}\chi^2; \quad \text{未知 } \sigma^2 \to t\text{/}\chi^2 \text{（自由度 } n-1\text{）}$$ $$\text{单样本均值} \to Z \text{（}\sigma\text{已知）或 } t(n-1)\text{（}\sigma\text{未知）}$$ $$\text{单样本方差} \to \chi^2(n-1) \quad \text{两样本方差比} \to F$$ $$\text{两样本均值差（等方差）} \to t(n_1+n_2-2)$$

7.9 独立性判定的"一步到位"法

考试策略：
  • 离散型：检查 $p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$ 是否对所有 $i,j$ 成立。只要有一对不成立，就不独立
  • 连续型：看 $f(x,y)$ 能否写成 $h_1(x) \cdot h_2(y)$ 的形式（即变量可分离）。若能，则独立；注意定义域必须也是矩形乘积
  • 二维正态：$\rho = 0 \Leftrightarrow$ 独立 ← 考试用得最多
