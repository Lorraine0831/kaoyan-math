1行列式

1.1 行列式性质

$$|A^T| = |A| \qquad |AB| = |A|\cdot|B| \qquad |kA| = k^n|A|$$ $$|A^{-1}| = \frac{1}{|A|} \qquad |A^*| = |A|^{n-1} \quad (A \text{ 为 } n \text{ 阶})$$ $$|A^m| = |A|^m \qquad |AB|=0 \Leftrightarrow |A|=0 \text{ 或 } |B|=0$$

1.2 展开定理

$$|A| = \sum_{k=1}^n a_{ik}A_{ik} \quad (\text{按第 } i \text{ 行展开})$$ $$|A| = \sum_{k=1}^n a_{kj}A_{kj} \quad (\text{按第 } j \text{ 列展开})$$ $$\sum_{k=1}^n a_{ik}A_{jk} = 0 \quad (i \neq j) \qquad \text{（异行（列）展开为0）}$$

1.3 特殊行列式

类型值
上/下三角行列式$a_{11}a_{22}\cdots a_{nn}$（主对角线元素之积）
对角行列式$a_{11}a_{22}\cdots a_{nn}$
2阶$\begin{vmatrix} a & b \\ c & d \end{vmatrix} = ad - bc$
3阶（沙路法）$a_1b_2c_3 + a_2b_3c_1 + a_3b_1c_2 - a_3b_2c_1 - a_1b_3c_2 - a_2b_1c_3$
范德蒙德行列式$\prod_{1\leq j < i \leq n}(x_i - x_j)$
分块上三角$\begin{vmatrix} A & C \\ O & B \end{vmatrix} = |A||B|$
分块下三角$\begin{vmatrix} A & O \\ C & B \end{vmatrix} = |A||B|$

1.4 克莱姆法则

若 $n$ 元线性方程组 $A\vec{x}=\vec{b}$ 的系数行列式 $|A| \neq 0$，则唯一解为：

$$x_j = \frac{D_j}{D}, \quad D = |A|, \quad D_j = \begin{vmatrix} a_{11} & \cdots & b_1 & \cdots & a_{1n} \\ \vdots & & \vdots & & \vdots \\ a_{n1} & \cdots & b_n & \cdots & a_{nn} \end{vmatrix}$$

其中 $D_j$ 是把 $A$ 的第 $j$ 列换成 $\vec{b}$ 后的行列式。

推论：齐次方程组 $A\vec{x}=\vec{0}$ 有非零解 $\Leftrightarrow$ $|A| = 0$。

2矩阵运算

2.1 基本运算法则

$$A + B = B + A \qquad (A+B)+C = A+(B+C)$$ $$(AB)C = A(BC) \qquad A(B+C) = AB + AC \qquad (B+C)A = BA + CA$$ $$k(AB) = (kA)B = A(kB) \qquad (AB)^T = B^T A^T$$ $$(A^T)^T = A \qquad (A+B)^T = A^T + B^T$$
注意：矩阵乘法不满足交换律（$AB \neq BA$ 一般情况），也不满足消去律（$AB=AC$ 且 $A\neq O$ 不能推出 $B=C$）。

2.2 转置与特殊矩阵

类型定义性质
对称矩阵$A^T = A$$a_{ij} = a_{ji}$
反对称矩阵$A^T = -A$$a_{ii}=0$, $a_{ij}=-a_{ji}$
正交矩阵$A^T A = I$$A^T = A^{-1}$, $|A|=\pm 1$
幂等矩阵$A^2 = A$特征值只能是 $0$ 或 $1$
对合矩阵$A^2 = I$$A^{-1} = A$

2.3 矩阵的幂

$$(AB)^k \neq A^k B^k \text{（一般情况）} \qquad (A^k)^l = A^{kl}$$ $$\text{若 } A = \alpha\beta^T \text{（秩1矩阵）} \;\Rightarrow\; A^n = (\beta^T\alpha)^{n-1} A$$

3逆矩阵与伴随矩阵

3.1 逆矩阵定义与判据

$$A \text{ 可逆} \;\Leftrightarrow\; \exists\, B, \; AB = BA = I \;\Leftrightarrow\; |A| \neq 0 \;\Leftrightarrow\; r(A) = n$$ $$A^{-1} = \frac{1}{|A|} A^* \quad (\text{伴随矩阵法})$$

3.2 逆矩阵的性质

$$(A^{-1})^{-1} = A \qquad (AB)^{-1} = B^{-1}A^{-1} \qquad (A^{-1})^T = (A^T)^{-1}$$ $$(kA)^{-1} = \frac{1}{k}A^{-1} \quad (k\neq 0) \qquad (A^*)^{-1} = \frac{A}{|A|} \quad (|A|\neq 0)$$ $$|A^{-1}| = |A|^{-1} \qquad (A^{-1})^* = \frac{A}{|A|}$$

3.3 伴随矩阵 $A^*$ 的核心性质

$$AA^* = A^*A = |A|\,I \qquad A^* = |A|\,A^{-1} \;\;(|A|\neq 0)$$ $$(A^*)^T = (A^T)^* \qquad (A^*)^* = |A|^{n-2}A \;\;(n\geq 2)$$ $$(AB)^* = B^*A^* \qquad (kA)^* = k^{n-1}A^*$$ $$|A^*| = |A|^{n-1} \qquad r(A^*) = \begin{cases} n & r(A)=n \\ 1 & r(A)=n-1 \\ 0 & r(A)\leq n-2 \end{cases}$$
高频考点：$r(A^*)$ 的三种情况是考试常考结论。$r(A)=n-1$ 时 $A^* \neq O$ 但 $|A^*|=0$。

3.4 分块矩阵的逆

$$\begin{pmatrix} A & O \\ O & B \end{pmatrix}^{-1} = \begin{pmatrix} A^{-1} & O \\ O & B^{-1} \end{pmatrix}$$ $$\begin{pmatrix} O & A \\ B & O \end{pmatrix}^{-1} = \begin{pmatrix} O & B^{-1} \\ A^{-1} & O \end{pmatrix}$$ $$\begin{pmatrix} A & C \\ O & B \end{pmatrix}^{-1} = \begin{pmatrix} A^{-1} & -A^{-1}CB^{-1} \\ O & B^{-1} \end{pmatrix}$$

4矩阵的秩

4.1 秩的重要不等式

$$r(AB) \leq \min\{r(A),\, r(B)\}$$ $$r(A+B) \leq r(A) + r(B)$$ $$r(A) + r(B) - n \leq r(AB) \quad (\text{西尔维斯特不等式})$$ $$\max\{r(A), r(B)\} \leq r\begin{pmatrix} A \\ B \end{pmatrix} \leq r(A) + r(B)$$

4.2 秩与可逆性

  • $A$ 为 $m\times n$ 矩阵，$r(A) = r \Leftrightarrow$ 存在 $r$ 阶非零子式，且所有 $r+1$ 阶子式为零
  • $r(A) = n \Leftrightarrow$ $A$ 可逆（$n$ 阶方阵）
  • 初等变换不改变秩
  • $r(A) = r(A^TA) = r(AA^T)$

4.3 初等矩阵

初等变换对应的初等矩阵行列式
交换 $i,j$ 行$E(i,j)$$-1$
第 $i$ 行乘 $k$$E(i(k))$$k$
第 $j$ 行加 $k$ 倍第 $i$ 行$E(ij(k))$$1$
$$A \xrightarrow{\text{左乘}} EA \quad A \xrightarrow{\text{右乘}} AE \qquad E^{-1} \text{ 仍为初等矩阵}$$

5向量

5.1 线性相关与线性无关

$$\vec{\alpha}_1, \ldots, \vec{\alpha}_m \text{ 线性相关} \;\Leftrightarrow\; \exists\, k_1,\ldots,k_m \text{ 不全为零}, \; \sum k_i\vec{\alpha}_i = \vec{0}$$ $$\text{线性无关} \;\Leftrightarrow\; \sum k_i\vec{\alpha}_i = \vec{0} \Rightarrow k_i = 0 \;\;\forall i$$
  • 单个向量 $\vec{\alpha}$ 线性相关 $\Leftrightarrow$ $\vec{\alpha}=\vec{0}$
  • 两个向量线性相关 $\Leftrightarrow$ 成比例
  • $m > n$ 时，$m$ 个 $n$ 维向量必线性相关
  • 部分相关 $\Rightarrow$ 整体相关；整体无关 $\Rightarrow$ 部分无关

5.2 内积、正交与施密特正交化

$$\langle \vec{\alpha}, \vec{\beta} \rangle = \vec{\alpha}^T\vec{\beta} = \sum a_i b_i \qquad \|\vec{\alpha}\| = \sqrt{\langle\vec{\alpha},\vec{\alpha}\rangle}$$ $$\vec{\alpha} \perp \vec{\beta} \;\Leftrightarrow\; \langle\vec{\alpha},\vec{\beta}\rangle = 0$$

施密特正交化：

$$\vec{\beta}_1 = \vec{\alpha}_1$$ $$\vec{\beta}_2 = \vec{\alpha}_2 - \frac{\langle\vec{\alpha}_2,\vec{\beta}_1\rangle}{\langle\vec{\beta}_1,\vec{\beta}_1\rangle}\vec{\beta}_1$$ $$\vec{\beta}_3 = \vec{\alpha}_3 - \frac{\langle\vec{\alpha}_3,\vec{\beta}_1\rangle}{\langle\vec{\beta}_1,\vec{\beta}_1\rangle}\vec{\beta}_1 - \frac{\langle\vec{\alpha}_3,\vec{\beta}_2\rangle}{\langle\vec{\beta}_2,\vec{\beta}_2\rangle}\vec{\beta}_2$$ $$\text{单位化：} \;\; \vec{e}_i = \frac{\vec{\beta}_i}{\|\vec{\beta}_i\|}$$

5.3 向量空间

$$\text{基} \to \text{维数} \to \text{坐标} \qquad \text{过渡矩阵 } P: (\vec{\beta}_1,\ldots,\vec{\beta}_n) = (\vec{\alpha}_1,\ldots,\vec{\alpha}_n)P$$ $$\text{坐标变换：} \;\; \vec{x}_{\text{新}} = P^{-1}\vec{x}_{\text{旧}}$$

6线性方程组

6.1 解的判据

对 $A\vec{x}=\vec{b}$（$A$ 为 $m\times n$）：

$$\text{无解} \;\Leftrightarrow\; r(A) \neq r(A|\vec{b})$$ $$\text{唯一解} \;\Leftrightarrow\; r(A) = r(A|\vec{b}) = n$$ $$\text{无穷多解} \;\Leftrightarrow\; r(A) = r(A|\vec{b}) < n$$ $$\text{齐次 } A\vec{x}=\vec{0} \text{ 有非零解} \;\Leftrightarrow\; r(A) < n$$

6.2 齐次方程组 $A\vec{x}=\vec{0}$ 的解结构

基础解系含 $n - r(A)$ 个线性无关的解向量 $\vec{\xi}_1, \ldots, \vec{\xi}_{n-r}$：

$$\vec{x} = c_1\vec{\xi}_1 + c_2\vec{\xi}_2 + \cdots + c_{n-r}\vec{\xi}_{n-r}$$ $$\text{解空间维数} = n - r(A)$$

6.3 非齐次方程组 $A\vec{x}=\vec{b}$ 的解结构

$$\vec{x} = \vec{\eta}^* + c_1\vec{\xi}_1 + \cdots + c_{n-r}\vec{\xi}_{n-r}$$

其中 $\vec{\eta}^*$ 为特解，$\vec{\xi}_i$ 为对应齐次方程组的基础解系。

性质：若 $\vec{\eta}_1, \vec{\eta}_2$ 是 $A\vec{x}=\vec{b}$ 的解，则 $\vec{\eta}_1-\vec{\eta}_2$ 是 $A\vec{x}=\vec{0}$ 的解。

6.4 公共解与同解

$$A\vec{x}=\vec{0} \text{ 与 } B\vec{x}=\vec{0} \text{ 同解} \;\Leftrightarrow\; r(A) = r(B) = r\!\begin{pmatrix}A\\B\end{pmatrix}$$

7特征值与特征向量

7.1 定义与求法

$$A\vec{\alpha} = \lambda\vec{\alpha} \;\;(\vec{\alpha}\neq\vec{0}) \qquad |\lambda I - A| = 0 \;\text{（特征方程）}$$ $$(\lambda I - A)\vec{\alpha} = \vec{0} \;\text{（求特征向量）}$$

7.2 特征值的性质

$$\sum_{i=1}^n \lambda_i = \text{tr}(A) = \sum a_{ii} \qquad \prod_{i=1}^n \lambda_i = |A|$$ $$\lambda_{A^T} = \lambda_A \qquad \lambda_{A^{-1}} = \frac{1}{\lambda_A} \qquad \lambda_{A^*} = \frac{|A|}{\lambda_A}$$ $$\lambda_{A^k} = \lambda_A^k \qquad \lambda_{A+kI} = \lambda_A + k \qquad \lambda_{f(A)} = f(\lambda_A)$$
注意：$\lambda_A = 0 \Leftrightarrow |A| = 0$（$A$ 不可逆）。不同特征值对应的特征向量线性无关。

7.3 相似与对角化

$$A \sim B \;\Leftrightarrow\; \exists\, P \text{ 可逆}, \;\; P^{-1}AP = B$$ $$A \text{ 可对角化} \;\Leftrightarrow\; \exists\, P, \;\; P^{-1}AP = \Lambda = \text{diag}(\lambda_1,\ldots,\lambda_n)$$ $$\Leftrightarrow\; A \text{ 有 } n \text{ 个线性无关的特征向量}$$ $$\Leftrightarrow\; \text{每个 } k_i \text{ 重特征值有 } k_i \text{ 个线性无关特征向量}$$

7.4 实对称矩阵

$$\text{实对称矩阵的特征值全为实数}$$ $$\text{不同特征值对应的特征向量正交}$$ $$\text{实对称矩阵一定可正交对角化}:\;\; Q^TAQ = \Lambda, \;\; Q^TQ = I$$

7.5 相似变换的性质

$$A \sim B \;\Rightarrow\; |A|=|B|,\; \text{tr}(A)=\text{tr}(B),\; r(A)=r(B),\; \lambda_A = \lambda_B$$ $$A \sim B \;\Rightarrow\; A^k \sim B^k,\; A^{-1}\sim B^{-1},\; A^T\sim B^T$$ $$A \sim B \;\Rightarrow\; f(A)\sim f(B)$$

8二次型

8.1 二次型的矩阵表示

$$f = \vec{x}^T A \vec{x}, \quad A \text{ 为对称矩阵}, \quad r(A) = r(f) \text{（二次型的秩）}$$

8.2 标准形与规范形

$$\text{标准形：} \;\; f = d_1 y_1^2 + d_2 y_2^2 + \cdots + d_n y_n^2$$ $$\text{规范形：} \;\; f = y_1^2 + \cdots + y_p^2 - y_{p+1}^2 - \cdots - y_{p+q}^2$$ $$p + q = r(f) \;\;(\text{秩}), \quad p - q = \text{符号差}$$ $$\text{惯性定理：} \;\; p, q \text{ 由 } f \text{ 唯一确定（合同不变量）}$$

8.3 正交变换化标准形

$$\text{实对称矩阵 } A \;\to\; \text{求特征值 } \lambda_i \;\to\; \text{正交化单位化特征向量} \;\to\; Q$$ $$Q^TAQ = \Lambda = \text{diag}(\lambda_1, \ldots, \lambda_n), \quad f = \lambda_1 y_1^2 + \cdots + \lambda_n y_n^2$$

8.4 配方法化标准形

步骤：逐步将含某变量的交叉项配成完全平方，每次消去一个变量，直到没有交叉项。

8.5 正定二次型

$$f = \vec{x}^T A \vec{x} \text{ 正定} \;\Leftrightarrow\; \forall\, \vec{x}\neq\vec{0},\; f > 0$$ $$\Leftrightarrow\; A \text{ 的特征值全为正}$$ $$\Leftrightarrow\; A \text{ 的各阶顺序主子式全为正}$$ $$\Leftrightarrow\; p = r = n \;\text{（正惯性指数 = 秩 = n）}$$ $$\Leftrightarrow\; A \text{ 合同于 } I_n$$

顺序主子式判据（最常用）：

$$\Delta_1 = a_{11} > 0, \;\; \Delta_2 = \begin{vmatrix} a_{11} & a_{12} \\ a_{21} & a_{22} \end{vmatrix} > 0, \;\; \ldots, \;\; \Delta_n = |A| > 0$$

8.6 合同关系

$$A \simeq B \;\Leftrightarrow\; \exists\, C \text{ 可逆}, \;\; C^TAC = B$$ $$A \simeq B \;\Rightarrow\; r(A)=r(B), \;\; \text{正负惯性指数相同}$$ $$\text{实对称矩阵 } A \simeq B \;\Leftrightarrow\; \text{正惯性指数、负惯性指数分别相等}$$

9高分技巧与易漏考点

9.1 $AB=O$ 的深层次理解

$$AB = O \;\Rightarrow\; \begin{cases} r(A) + r(B) \leq n \\ B \text{ 的每一列都是 } A\vec{x}=\vec{0} \text{ 的解} \\ B \text{ 的列向量空间 } \subseteq \text{Ker}(A) \end{cases}$$
做题思路：$AB=O$ 不是"$A,B$ 为零"的意思——而是 $B$ 的非零列是 $A\vec{x}=\vec{0}$ 的非零解向量。这给出了解向量的来源。

9.2 齐次方程组同解的条件（完善版）

$$Ax=0 \text{ 与 } Bx=0 \text{ 同解} \;\Leftrightarrow\; r(A)=r(B)=r\!\begin{pmatrix}A\\B\end{pmatrix}$$ $$\Leftrightarrow\; A, B \text{ 的行向量组等价}$$ $$\Leftrightarrow\; A \text{ 的行可由 } B \text{ 的行线性表出，且反之亦然}$$

9.3 最小二乘解与正规方程

$$A\vec{x}=\vec{b} \text{ 无解时，求其最小二乘解：} \;\; A^TA\vec{x} = A^T\vec{b}$$ $$\text{该方程必有解，且最小二乘解使 } \|A\vec{x}-\vec{b}\|^2 \text{ 最小}$$
背景：$A^TA$ 是正定或半正定的，因此 $A^TA\vec{x}=A^T\vec{b}$ 总相容。这在线性回归中用得非常多。

9.4 秩1矩阵的速判与速算

$$\text{若 } r(A)=1, \;\text{则 } A = \vec{u}\vec{v}^T, \;\; A^n = (\vec{v}^T\vec{u})^{n-1} A = (\text{tr}(A))^{n-1} A$$ $$\text{特征值：} \;\; \lambda_1 = \text{tr}(A), \;\; \lambda_2 = \cdots = \lambda_n = 0$$
注意：秩1矩阵的迹 $=\vec{v}^T\vec{u}$。直接用 $A^n=(\text{tr}A)^{n-1}A$，比对角化快得多。

9.5 实对称矩阵一个常被忽视的性质

$$A \text{ 实对称且 } A^k = O \;\Rightarrow\; A = O \quad (\text{特别是 } A^2=O \Rightarrow A=O)$$ $$A \text{ 实对称且正定} \;\Rightarrow\; A^{-1} \text{ 也正定，} A^* \text{ 也正定}$$
应用：证 $A=O$ 常走"先证 $A^2=O$，如果 $A$ 还是实对称则完事"。一般矩阵 $A^2=O$ 不能推出 $A=O$。

9.6 正交矩阵特征值的三个结论

$$A \text{ 正交矩阵} \;\Rightarrow\; |\lambda| = 1$$ $$\text{实正交矩阵的复特征值成对出现：} \;\; e^{\pm i\theta}$$ $$\text{奇数阶实正交矩阵必有特征值 } \lambda = 1 \text{ 或 } -1$$

9.7 "主对角占优"快速判断行列式非零

技巧：考试中看到严格对角占优矩阵（每行主对角线绝对值 > 同行的其他所有元素绝对值之和），直接判可逆（$|A|\neq 0$），不需计算行列式。

9.8 二次型与特征值的"翻译"关系

$$\lambda_{\min}(A) \cdot \|\vec{x}\|^2 \leq \vec{x}^T A\vec{x} \leq \lambda_{\max}(A) \cdot \|\vec{x}\|^2 \quad (A \text{ 实对称})$$ $$\max_{\|\vec{x}\|=1} \vec{x}^T A\vec{x} = \lambda_{\max}, \qquad \min_{\|\vec{x}\|=1} \vec{x}^T A\vec{x} = \lambda_{\min}$$
雷利商：$\frac{\vec{x}^T A\vec{x}}{\vec{x}^T \vec{x}}$ 在特征向量处取极值——证明题中非常有用。

9.9 求解矩阵方程的"降维"策略

高频套路：已知 $A$ 的特征值和特征向量，求 $B = f(A)$ 的特征和性质。口诀——"特征值走代数运算，特征向量原封不动"。
$$A\vec{\alpha} = \lambda\vec{\alpha} \;\Rightarrow\; f(A)\vec{\alpha} = f(\lambda)\vec{\alpha}$$
