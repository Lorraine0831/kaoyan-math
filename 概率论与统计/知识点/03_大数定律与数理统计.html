1切比雪夫不等式

$$P(|X - E(X)| \geq \varepsilon) \leq \frac{D(X)}{\varepsilon^2}$$
意义：不论 $X$ 服从什么分布，只要知道期望和方差，就能给出偏离期望的概率上界。
应用：用于证明大数定律，以及在不完全知道分布时给出概率估计。取 $\varepsilon = k\sigma$ 得 $P(|X-\mu| \geq k\sigma) \leq \frac{1}{k^2}$。

2大数定律

2.1 切比雪夫大数定律

设 $X_1, X_2, \ldots$ 相互独立，$E(X_i) = \mu_i$，$D(X_i) \leq C$（一致有界），则

$$\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \frac{1}{n}\sum_{i=1}^n \mu_i$$

2.2 辛钦大数定律

设 $X_1, \ldots, X_n$ 独立同分布，$E(X_i) = \mu$，则

$$\frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} \mu$$
意义：样本均值依概率收敛于总体均值。这是矩估计的理论依据。

2.3 伯努利大数定律

设 $n_A$ 为 $n$ 次独立试验中事件 $A$ 发生的次数，$P(A) = p$，则

$$\frac{n_A}{n} \xrightarrow{P} p$$
意义：频率依概率收敛于概率。这是"频率代替概率"的理论依据。

3中心极限定理

3.1 林德伯格-列维定理

设 $X_1, \ldots, X_n$ 独立同分布，$E(X_i) = \mu$，$D(X_i) = \sigma^2 > 0$，则

$$\frac{\sum_{i=1}^n X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1)$$ $$\text{即} \;\; \sum_{i=1}^n X_i \;\dot{\sim}\; N(n\mu, \; n\sigma^2), \qquad \bar{X} \;\dot{\sim}\; N\!\left(\mu, \;\frac{\sigma^2}{n}\right)$$
意义：无论原始分布是什么，大量独立同分布变量之和近似服从正态分布。

3.2 棣莫弗-拉普拉斯定理

设 $X \sim B(n, p)$，则

$$\frac{X - np}{\sqrt{np(1-p)}} \;\dot{\sim}\; N(0,1)$$

这是林德伯格-列维定理对二项分布的特例（$X = \sum X_i$，$X_i \sim B(1,p)$）。

3.3 应用：近似计算

$$P(a \leq \bar{X} \leq b) \approx \Phi\!\left(\frac{b - \mu}{\sigma/\sqrt{n}}\right) - \Phi\!\left(\frac{a - \mu}{\sigma/\sqrt{n}}\right)$$

4数理统计基本概念

4.1 基本概念

  • 总体：研究对象的全体（一个随机变量 $X$）
  • 样本：从总体中抽取的 $n$ 个独立同分布变量 $X_1, \ldots, X_n$
  • 统计量：样本的不含未知参数的函数 $g(X_1, \ldots, X_n)$
  • 样本均值：$\bar{X} = \frac{1}{n}\sum X_i$
  • 样本方差：$S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$（注意分母是 $n-1$）
  • 样本标准差：$S = \sqrt{S^2}$

4.2 三大抽样分布

$\chi^2$ 分布

$$Z_1, \ldots, Z_n \sim N(0,1) \text{ 独立} \;\Rightarrow\; \chi^2 = \sum Z_i^2 \sim \chi^2(n)$$ $$E(\chi^2(n)) = n, \quad D(\chi^2(n)) = 2n$$ $$\chi_1^2 \sim \chi^2(n_1), \;\chi_2^2 \sim \chi^2(n_2), \;\text{独立} \;\Rightarrow\; \chi_1^2 + \chi_2^2 \sim \chi^2(n_1 + n_2) \;\text{（可加性）}$$

$t$ 分布

$$Z \sim N(0,1), \;\; V \sim \chi^2(n), \;\; Z, V \text{ 独立} \;\Rightarrow\; T = \frac{Z}{\sqrt{V/n}} \sim t(n)$$ $$E(T) = 0 \;(n > 1), \quad D(T) = \frac{n}{n-2} \;(n > 2)$$ $$t(n) \xrightarrow{n \to \infty} N(0,1) \qquad \text{密度关于 } 0 \text{ 对称}$$

$F$ 分布

$$U \sim \chi^2(m), \;\; V \sim \chi^2(n), \;\; U, V \text{ 独立} \;\Rightarrow\; F = \frac{U/m}{V/n} \sim F(m, n)$$ $$\frac{1}{F} \sim F(n, m) \qquad E(F) = \frac{n}{n-2} \;(n > 2)$$

4.3 正态总体的抽样分布

设 $X_1, \ldots, X_n \sim N(\mu, \sigma^2)$ 独立，$\bar{X}$ 为样本均值，$S^2$ 为样本方差：

$$\bar{X} \sim N\!\left(\mu, \;\frac{\sigma^2}{n}\right) \;\;\Rightarrow\;\; \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$$ $$\frac{(n-1)S^2}{\sigma^2} \sim \chi^2(n-1)$$ $$\frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$$ $$\bar{X} \text{ 与 } S^2 \text{ 独立}$$

两个独立正态总体 $N(\mu_1, \sigma_1^2)$ 和 $N(\mu_2, \sigma_2^2)$：

$$\frac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1 - 1, \; n_2 - 1) \;\;(\sigma_1^2 = \sigma_2^2 \text{ 时})$$ $$\frac{(\bar{X}_1 - \bar{X}_2) - (\mu_1 - \mu_2)}{S_w\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}} \sim t(n_1+n_2-2), \;\; S_w^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2}$$

5参数估计

5.1 矩估计法

用样本矩替代总体矩，解出参数估计。

$$\text{令 } E(X) = \bar{X} = \frac{1}{n}\sum X_i \;\text{（一阶矩）}$$ $$\text{令 } E(X^2) = \frac{1}{n}\sum X_i^2 \;\text{（二阶矩）}$$ $$\text{解方程组得 } \hat{\theta}_1, \hat{\theta}_2$$

5.2 最大似然估计

构造似然函数（样本取到当前值的"似然度"），取使其最大的参数值。

$$L(\theta) = \prod_{i=1}^n f(x_i; \theta) \quad \text{（连续型）或} \quad L(\theta) = \prod P(X = x_i; \theta) \quad \text{（离散型）}$$ $$\ln L(\theta) = \sum_{i=1}^n \ln f(x_i; \theta)$$ $$\frac{\partial \ln L}{\partial \theta} = 0 \;\Rightarrow\; \hat{\theta}_{MLE}$$
步骤：构造 $L$ → 取对数 $\ln L$ → 求导令其为零 → 解出 $\hat{\theta}$。若导数无法为零，则看单调性取边界。

5.3 估计量的评价标准

标准定义说明
无偏性$E(\hat{\theta}) = \theta$估计量的期望等于真值
有效性$D(\hat{\theta}_1) < D(\hat{\theta}_2)$在无偏估计中，方差越小越有效
一致性（相合性）$\hat{\theta}_n \xrightarrow{P} \theta$样本量增大时估计值收敛于真值
重要结论：$S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$ 是 $\sigma^2$ 的无偏估计，而 $\frac{1}{n}\sum(X_i-\bar{X})^2$ 是有偏的。这就是样本方差分母为 $n-1$ 的原因。

5.4 区间估计

构造统计量，利用抽样分布给出包含参数的置信区间：

参数条件置信区间（置信度 $1-\alpha$）
$\mu$$\sigma^2$ 已知$\bar{X} \pm z_{\alpha/2}\frac{\sigma}{\sqrt{n}}$
$\mu$$\sigma^2$ 未知$\bar{X} \pm t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}}$
$\sigma^2$$\mu$ 未知$\left(\frac{(n-1)S^2}{\chi^2_{\alpha/2}(n-1)}, \;\frac{(n-1)S^2}{\chi^2_{1-\alpha/2}(n-1)}\right)$

6假设检验

6.1 基本概念

  • 原假设 $H_0$ 与备择假设 $H_1$
  • 检验统计量：在 $H_0$ 成立时分布已知的统计量
  • 拒绝域 $W$：使 $H_0$ 被拒绝的统计量取值范围
  • 显著性水平 $\alpha$：$P(\text{拒绝}H_0 | H_0 \text{ 真}) = \alpha$

6.2 两类错误

错误类型含义概率
第一类（弃真）$H_0$ 真但拒绝 $H_0$$\alpha$
第二类（取伪）$H_0$ 假但接受 $H_0$$\beta$

在样本量固定时，减小 $\alpha$ 会增大 $\beta$，反之亦然。通常先控制 $\alpha$。

6.3 正态总体参数检验

检验$H_0$统计量（$\sigma^2$已知）统计量（$\sigma^2$未知）
单样本均值$\mu = \mu_0$$Z = \frac{\bar{X}-\mu_0}{\sigma/\sqrt{n}}$$T = \frac{\bar{X}-\mu_0}{S/\sqrt{n}}$
单样本方差$\sigma^2 = \sigma_0^2$$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2}$
$$\text{拒绝域（双侧，}\sigma^2\text{未知）：} \;\; |T| > t_{\alpha/2}(n-1)$$ $$\text{拒绝域（单侧）：} \;\; T > t_\alpha(n-1) \;\;\text{或}\;\; T < -t_\alpha(n-1)$$
