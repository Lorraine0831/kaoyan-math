1向量的线性相关性

1.1 线性组合与线性表出

设 $\vec{\alpha}_1, \ldots, \vec{\alpha}_m$ 为一组向量，$k_1\vec{\alpha}_1 + \cdots + k_m\vec{\alpha}_m$ 称为它们的线性组合。若 $\vec{\beta} = k_1\vec{\alpha}_1 + \cdots + k_m\vec{\alpha}_m$，则称 $\vec{\beta}$ 可由 $\vec{\alpha}_1, \ldots, \vec{\alpha}_m$ 线性表出。

$$\vec{\beta} \text{ 可由 } \vec{\alpha}_1, \ldots, \vec{\alpha}_m \text{ 线性表出} \;\Leftrightarrow\; r(\vec{\alpha}_1, \ldots, \vec{\alpha}_m) = r(\vec{\alpha}_1, \ldots, \vec{\alpha}_m, \vec{\beta})$$

1.2 线性相关与线性无关

$$\vec{\alpha}_1, \ldots, \vec{\alpha}_m \text{ 线性相关} \;\Leftrightarrow\; \exists\, k_1, \ldots, k_m \text{ 不全为零}, \;\sum_{i=1}^m k_i\vec{\alpha}_i = \vec{0}$$ $$\text{线性无关} \;\Leftrightarrow\; \sum k_i\vec{\alpha}_i = \vec{0} \;\Rightarrow\; k_i = 0 \;\forall\, i$$

1.3 重要结论

  • 单个向量 $\vec{\alpha}$ 线性相关 $\Leftrightarrow$ $\vec{\alpha} = \vec{0}$
  • 两个向量线性相关 $\Leftrightarrow$ 成比例（共线）
  • $m > n$ 时，$m$ 个 $n$ 维向量必线性相关
  • 部分相关 $\Rightarrow$ 整体相关；整体无关 $\Rightarrow$ 部分无关
  • 低维无关 $\Rightarrow$ 增加分量后高维无关（"接长"）；高维相关 $\Rightarrow$ 去掉分量后低维相关（"截短"）
  • 若 $\vec{\alpha}_1, \ldots, \vec{\alpha}_r$ 线性无关，而 $\vec{\alpha}_1, \ldots, \vec{\alpha}_r, \vec{\beta}$ 线性相关，则 $\vec{\beta}$ 可由 $\vec{\alpha}_1, \ldots, \vec{\alpha}_r$ 唯一表出

1.4 判定方法

方法判据
定义法设 $\sum k_i\vec{\alpha}_i = \vec{0}$，解齐次方程组，有非零解 $\Leftrightarrow$ 相关
秩法$r(\vec{\alpha}_1, \ldots, \vec{\alpha}_m) < m \Leftrightarrow$ 线性相关；$= m \Leftrightarrow$ 线性无关
行列式法$m = n$（向量个数=维数）时，$|A| = 0 \Leftrightarrow$ 相关

2极大线性无关组与秩

2.1 定义

向量组 $T$ 中的一个部分组 $\vec{\alpha}_1, \ldots, \vec{\alpha}_r$ 称为极大线性无关组，若：

  1. $\vec{\alpha}_1, \ldots, \vec{\alpha}_r$ 线性无关
  2. $T$ 中任一向量可由 $\vec{\alpha}_1, \ldots, \vec{\alpha}_r$ 线性表出

极大无关组中向量的个数 $r$ 称为该向量组的秩。极大无关组不唯一，但秩唯一。

2.2 求极大无关组

将向量组排成矩阵，做初等行变换化为行阶梯形，主元所在列对应的原向量即为极大无关组。

2.3 向量组的等价

$$\text{向量组 } A \text{ 与 } B \text{ 等价} \;\Leftrightarrow\; A \text{ 与 } B \text{ 可互相线性表出} \;\Leftrightarrow\; r(A) = r(B) = r(A|B)$$

3内积与正交

3.1 内积

$$\langle\vec{\alpha}, \vec{\beta}\rangle = \vec{\alpha}^T\vec{\beta} = \sum_{i=1}^n a_i b_i \qquad \|\vec{\alpha}\| = \sqrt{\langle\vec{\alpha},\vec{\alpha}\rangle}$$ $$\langle\vec{\alpha}, \vec{\beta}\rangle = 0 \;\Leftrightarrow\; \vec{\alpha} \perp \vec{\beta}$$

3.2 施密特正交化

将线性无关向量组 $\vec{\alpha}_1, \ldots, \vec{\alpha}_m$ 化为正交向量组 $\vec{\beta}_1, \ldots, \vec{\beta}_m$：

$$\vec{\beta}_1 = \vec{\alpha}_1$$ $$\vec{\beta}_2 = \vec{\alpha}_2 - \frac{\langle\vec{\alpha}_2, \vec{\beta}_1\rangle}{\langle\vec{\beta}_1, \vec{\beta}_1\rangle}\vec{\beta}_1$$ $$\vec{\beta}_k = \vec{\alpha}_k - \sum_{i=1}^{k-1}\frac{\langle\vec{\alpha}_k, \vec{\beta}_i\rangle}{\langle\vec{\beta}_i, \vec{\beta}_i\rangle}\vec{\beta}_i$$ $$\text{单位化：} \;\; \vec{e}_k = \frac{\vec{\beta}_k}{\|\vec{\beta}_k\|}$$

3.3 正交矩阵

$$A^TA = I \;\Leftrightarrow\; A \text{ 为正交矩阵} \;\Leftrightarrow\; \text{行（列）向量组为正交单位向量组}$$ $$A^{-1} = A^T \qquad |A| = \pm 1$$

4线性方程组解的判据

4.1 非齐次方程组 $A\vec{x} = \vec{b}$

设 $A$ 为 $m \times n$ 矩阵，增广矩阵 $\bar{A} = (A|\vec{b})$：

$$\text{无解} \;\Leftrightarrow\; r(A) < r(\bar{A})$$ $$\text{唯一解} \;\Leftrightarrow\; r(A) = r(\bar{A}) = n$$ $$\text{无穷多解} \;\Leftrightarrow\; r(A) = r(\bar{A}) < n$$

4.2 齐次方程组 $A\vec{x} = \vec{0}$

$$\text{只有零解} \;\Leftrightarrow\; r(A) = n$$ $$\text{有非零解} \;\Leftrightarrow\; r(A) < n \;\Leftrightarrow\; |A| = 0 \text{（方阵时）}$$

5齐次方程组解的结构

5.1 解空间的性质

  • 若 $\vec{\xi}_1, \vec{\xi}_2$ 是 $A\vec{x}=\vec{0}$ 的解，则 $k_1\vec{\xi}_1 + k_2\vec{\xi}_2$ 也是解
  • 解集构成向量空间，维数 $= n - r(A)$

5.2 基础解系

基础解系是解空间的一个基，含 $n - r(A)$ 个线性无关的解向量 $\vec{\xi}_1, \ldots, \vec{\xi}_{n-r}$。

$$\vec{x} = c_1\vec{\xi}_1 + c_2\vec{\xi}_2 + \cdots + c_{n-r}\vec{\xi}_{n-r} \quad (c_i \in \mathbb{R})$$
求法：将 $A$ 化为行最简形，令自由变量分别取 $(1,0,\ldots,0)^T, (0,1,\ldots,0)^T, \ldots$，回代得基础解系。

6非齐次方程组解的结构

6.1 通解公式

$$\vec{x} = \vec{\eta}^* + c_1\vec{\xi}_1 + c_2\vec{\xi}_2 + \cdots + c_{n-r}\vec{\xi}_{n-r}$$

其中 $\vec{\eta}^*$ 是 $A\vec{x}=\vec{b}$ 的一个特解，$\vec{\xi}_i$ 是对应齐次方程组 $A\vec{x}=\vec{0}$ 的基础解系。

6.2 解的性质

  • 若 $\vec{\eta}_1, \vec{\eta}_2$ 是 $A\vec{x}=\vec{b}$ 的解，则 $\vec{\eta}_1 - \vec{\eta}_2$ 是 $A\vec{x}=\vec{0}$ 的解
  • 若 $\vec{\eta}$ 是 $A\vec{x}=\vec{b}$ 的解，$\vec{\xi}$ 是 $A\vec{x}=\vec{0}$ 的解，则 $\vec{\eta} + \vec{\xi}$ 是 $A\vec{x}=\vec{b}$ 的解
  • 若 $A\vec{x}=\vec{b}$ 有 $\vec{\eta}_1, \ldots, \vec{\eta}_{r+1}$ 个线性无关的解，则 $r + 1 = n - r(A) + 1$

6.3 公共解与同解

$$A\vec{x}=\vec{0} \text{ 与 } B\vec{x}=\vec{0} \text{ 同解} \;\Leftrightarrow\; r(A) = r(B) = r\!\begin{pmatrix}A \\ B\end{pmatrix}$$ $$A\vec{x}=\vec{0} \text{ 的解都是 } B\vec{x}=\vec{0} \text{ 的解} \;\Leftrightarrow\; r(A) = r\!\begin{pmatrix}A \\ B\end{pmatrix}$$
