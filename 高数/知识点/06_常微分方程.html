1基本概念

1.1 微分方程的分类

分类标准类型示例
自变量个数常微分方程 / 偏微分方程$y' + xy = 0$ / $u_x + u_y = 0$
最高阶数一阶 / 二阶 / $n$ 阶$y'' + y = 0$（二阶）
未知函数及导数次数线性 / 非线性$y'' + py' + qy = 0$（线性）
系数是否常数常系数 / 变系数$y'' + 4y = 0$（常系数）

1.2 解的概念

  • 通解：含独立任意常数的个数等于方程阶数的解
  • 特解：通解中给常数确定值后得到的解
  • 初值问题（柯西问题）：求满足初始条件的特解

2一阶微分方程

2.1 可分离变量的方程

标准形式：$y' = f(x)g(y)$

$$\frac{dy}{g(y)} = f(x)\,dx \;\Rightarrow\; \int \frac{dy}{g(y)} = \int f(x)\,dx + C$$

2.2 齐次方程

标准形式：$y' = \varphi\!\left(\frac{y}{x}\right)$

$$\text{令 } u = \frac{y}{x}, \;\; y = ux, \;\; y' = u + xu'$$ $$\Rightarrow \; u + xu' = \varphi(u) \;\Rightarrow\; \frac{du}{\varphi(u)-u} = \frac{dx}{x}$$

化为可分离变量方程后求解。

2.3 一阶线性微分方程

标准形式：$y' + P(x)y = Q(x)$

$$y = e^{-\int P\,dx}\left[\int Q\,e^{\int P\,dx}\,dx + C\right]$$
推导：乘以积分因子 $\mu = e^{\int P\,dx}$，方程变为 $(y\cdot\mu)' = Q\cdot\mu$，再积分即得。

2.4 伯努利方程

标准形式：$y' + P(x)y = Q(x)\,y^n$ $\;(n \neq 0,1)$

$$\text{令 } z = y^{1-n} \;\Rightarrow\; z' + (1-n)P(x)z = (1-n)Q(x)$$

化为关于 $z$ 的一阶线性方程。

2.5 全微分方程（恰当方程）

若 $P\,dx + Q\,dy = 0$ 满足 $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$，则存在 $u(x,y)$ 使 $du = P\,dx + Q\,dy$，通解为

$$u(x,y) = C$$

若不满足恰当条件，可找积分因子 $\mu(x,y)$ 使 $\mu P\,dx + \mu Q\,dy = 0$ 变为恰当方程。

3可降阶的高阶方程

3.1 $y^{(n)} = f(x)$ 型

方程中不含 $y$ 和低阶导数，直接逐次积分：

$$y^{(n-1)} = \int f(x)\,dx + C_1 \;\Rightarrow\; \text{逐次积分 } n \text{ 次}$$

3.2 缺 $y$ 型：$y'' = f(x, y')$

$$\text{令 } p = y', \;\; y'' = p' \;\Rightarrow\; p' = f(x, p)$$

这是关于 $p$ 和 $x$ 的一阶方程，解出 $p$ 后再积分得 $y$。

3.3 缺 $x$ 型：$y'' = f(y, y')$

$$\text{令 } p = y', \;\; y'' = \frac{dp}{dx} = \frac{dp}{dy}\cdot\frac{dy}{dx} = p\,\frac{dp}{dy}$$ $$\Rightarrow\; p\frac{dp}{dy} = f(y, p)$$

这是关于 $p$ 和 $y$ 的一阶方程。注意这里视 $y$ 为自变量。

易错点：缺 $x$ 型中 $y'' = p\frac{dp}{dy}$，不是 $p'$。因为现在 $p$ 是 $y$ 的函数，需用链式法则转换。

4线性微分方程解的结构

4.1 齐次线性方程的叠加原理

若 $y_1, y_2$ 是 $y'' + p(x)y' + q(x)y = 0$ 的解，则 $C_1 y_1 + C_2 y_2$ 也是解。

4.2 线性无关与朗斯基行列式

两个解 $y_1, y_2$ 线性无关 $\Leftrightarrow$ $W(y_1, y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} \neq 0$

4.3 齐次通解

$$\bar{y} = C_1 y_1 + C_2 y_2 \quad (y_1, y_2 \text{ 线性无关})$$

4.4 非齐次通解结构

$$y = \bar{y} + y^* = (\text{齐次通解}) + (\text{非齐次特解})$$

4.5 叠加原理（非齐次）

若 $y_1^*$ 是 $Ly = f_1$ 的特解，$y_2^*$ 是 $Ly = f_2$ 的特解，则 $y_1^* + y_2^*$ 是 $Ly = f_1 + f_2$ 的特解。

5二阶常系数线性齐次方程

5.1 解法：特征方程法

对方程 $y'' + py' + qy = 0$，写出特征方程：

$$r^2 + pr + q = 0$$

5.2 三种情况

判别式特征根通解
$\Delta = p^2 - 4q > 0$$r_{1,2} = \frac{-p\pm\sqrt{\Delta}}{2}$（不等实根）$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$
$\Delta = 0$$r_1 = r_2 = -\frac{p}{2}$（重根）$y = (C_1 + C_2 x)e^{rx}$
$\Delta < 0$$r = \alpha \pm \beta i$，$\alpha = -\frac{p}{2}$，$\beta = \frac{\sqrt{-\Delta}}{2}$$y = e^{\alpha x}(C_1 \cos\beta x + C_2 \sin\beta x)$

5.3 高阶推广

对于 $n$ 阶常系数齐次方程，同样求特征方程 $r$ 的根：

  • 单实根 $r$：对应 $Ce^{rx}$
  • $k$ 重实根 $r$：对应 $(C_0 + C_1 x + \cdots + C_{k-1}x^{k-1})e^{rx}$
  • 单共轭复根 $\alpha\pm\beta i$：对应 $e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$
  • $k$ 重共轭复根：乘以 $x$ 的多项式

6二阶常系数线性非齐次方程

6.1 待定系数法

对方程 $y'' + py' + qy = f(x)$，根据 $f(x)$ 的类型设特解形式：

6.2 类型一：$f(x) = P_m(x)\,e^{\lambda x}$

其中 $P_m(x)$ 为 $m$ 次多项式。设

$$y^* = x^k \, Q_m(x)\, e^{\lambda x}$$ $$k = \begin{cases} 0 & \lambda \text{ 不是特征根} \\ 1 & \lambda \text{ 是单特征根} \\ 2 & \lambda \text{ 是二重特征根} \end{cases}$$

$Q_m(x)$ 是与 $P_m$ 同次的待定多项式。

6.3 类型二：$f(x) = e^{\alpha x}[P_l(x)\cos\beta x + P_n(x)\sin\beta x]$

$$y^* = x^k \, e^{\alpha x}[Q_m(x)\cos\beta x + R_m(x)\sin\beta x]$$ $$m = \max(l, n), \quad k = \begin{cases} 0 & \alpha\pm\beta i \text{ 不是特征根} \\ 1 & \alpha\pm\beta i \text{ 是单特征根} \end{cases}$$

6.4 特解形式总结表

$f(x)$ 的形式判断 $\lambda$（或 $\alpha\pm\beta i$）$k$ 值特解待定形式
$P_m(x)e^{\lambda x}$$\lambda$ 非特征根0$Q_m(x)e^{\lambda x}$
$\lambda$ 单根1$x\,Q_m(x)e^{\lambda x}$
$\lambda$ 二重根2$x^2\,Q_m(x)e^{\lambda x}$
$e^{\alpha x}[\cdots\cos\beta x + \cdots\sin\beta x]$$\alpha\pm\beta i$ 非特征根0$e^{\alpha x}[Q_m\cos\beta x + R_m\sin\beta x]$
$\alpha\pm\beta i$ 单根1$x\,e^{\alpha x}[Q_m\cos\beta x + R_m\sin\beta x]$
常见错误：
  • 忘记乘 $x^k$：当 $\lambda$ 是特征根时必须乘 $x^k$，否则特解形式与齐次解重复
  • 多项式次数设错：$Q_m$ 必须是完整 $m$ 次多项式，不能漏项
  • 三角函数型必须同时设 $\cos$ 和 $\sin$，即使 $f(x)$ 只有其中一项

6.5 常数变易法（一般方法）

若 $f(x)$ 不属于上述两类，可用常数变易法。已知齐次通解 $\bar{y} = C_1 y_1 + C_2 y_2$，设

$$y^* = C_1(x)\,y_1 + C_2(x)\,y_2$$ $$\text{解方程组：} \begin{cases} C_1'y_1 + C_2'y_2 = 0 \\ C_1'y_1' + C_2'y_2' = f(x) \end{cases}$$

求出 $C_1'(x), C_2'(x)$ 后积分即得 $C_1(x), C_2(x)$。

7欧拉方程

7.1 标准形式

$$x^n y^{(n)} + p_1 x^{n-1} y^{(n-1)} + \cdots + p_{n-1}xy' + p_n y = f(x)$$

特点：各项系数是 $x$ 的幂次，且幂次与导数阶数一致。

7.2 解法：欧拉代换

$$\text{令 } x = e^t \;\;(\text{即 } t = \ln|x|)$$ $$x\frac{d}{dx} = \frac{d}{dt} \;\equiv\; D, \quad x^2\frac{d^2}{dx^2} = D(D-1), \quad \cdots$$ $$x^k\frac{d^k}{dx^k} = D(D-1)(D-2)\cdots(D-k+1)$$

代入后化为常系数线性方程，解出关于 $t$ 的通解后，回代 $t = \ln|x|$ 即可。
